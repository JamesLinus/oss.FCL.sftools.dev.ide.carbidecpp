Tutorial: EqualContainer - Create UI Design for UI Designer Project

Create and/or select an existing UI Designer project in the Project Explorer view. Select File > New > S60 UI Design to display the S60 UI Design Selection wizard. The selected project will appear in the Add to project: field. Click Next to display a list of S60 user interface designs; as shown in Figure 1.

Figure 1. Equal Container Visible in UI Design Wizard

The new custom container is visible in the wizard. Select the Equal Container and click Next. The Container Name and Type dialog will appear. Enter a name for the new container type or accept the default entry; as shown in Figure 2.

Figure 2. Container Name and Type

Click Finish to add the new S60 UI design to the selected project. The example projectNameEqualContainer.uidesign file will appear in the project in the Project Explorer view.
