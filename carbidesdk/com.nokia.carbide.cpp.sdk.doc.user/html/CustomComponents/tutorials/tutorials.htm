Custom Component Tutorials

Three tutorials are provided to help you in creating your own custom components. One tutorial is provided for creating a top-level container component S60 and another is provided for S60 and UIQ that demostrates how to create a label component that can be used in a UI design and draws the text vertically.

Example files for the tutorials are provided in a zip file in the s60 components plugin located at: {Carbide_Installation_Dir}\plugins\com.nokia.sdt.series60.componentlibrary_version \templates\tutorials\example_components\example_components.zip, where version is the current version of UI Designer. The example files can be unzipped into a new General project that you create; as explained in the following tutorials.

  • Equal Container S60
  • Vertical Label S60
  • Vertical Label UIQ
