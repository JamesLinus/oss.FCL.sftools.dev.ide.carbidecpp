Working with Objects

You can add objects to your existing UI designs by dragging them from the UI Design Editor view's Palette into the layout or non-layout regions.

After adding objects you are able to perform the following actions:

  • Selecting objects
  • Resizing objects

Selecting Objects

In the UI Design Editor view :

  1. Click on an object

    A bounding rectangle appears around the object (Figure 1), the same object is highlighted in the Outline view, and its properties are shown in the Properties view.

  2. To add additional objects to the selection:
    • Shift-click or control-click an object to select it and all objects between the first selected object and the second

      Another bounding rectangle appears around the newly selected object (Figure 1).

      Figure 1- Multiple objects selected

       

NOTE The properties of the first object selected continue to be shown in the Properties view.

 

Resizing Objects

Object's size is determined the first time that it is added to the layout, and is defined by the row layout manager (Figure 2). It is not possible to re-fit the size later. Depending on the original size of the object (for example an image), the row layout manager calculates the amount of rows for the object.

Figure 2 - Object added and auto sized by the row layout manager
