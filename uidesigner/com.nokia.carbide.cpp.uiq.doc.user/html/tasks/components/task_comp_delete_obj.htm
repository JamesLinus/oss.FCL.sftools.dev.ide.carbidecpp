Deleting Objects

You may delete objects from your existing UI Designs.

Figure 1 - A selected object

Deleting an object from a UI Design.

  1. Select one or more objects in an open UI Design.
  2. Choose one of the following methods to delete the selected objects:
    • Right-click on the selected object(s) and select the Delete option from the context menu, OR
    • Select the Delete option from the Edit menu, OR
    • Press the Delete key

NOTE Deletion of objects is Undoable.
