Setting the Project Name

The New Project wizard guides you through the tasks needed to create a new UIQ C++ GUI project for use with the UI Designer. Each page of the wizard allows you to customize the new project to meet your specific needs.

Use the Symbian OS C++ Project page of the New Project Wizard window to name the new project and specify where to store project files.

Figure 1 - Naming the project

Naming the New Project

  1. Enter a unique project name in the Project name field (Figure 1)
  2. Specify where to store Project related files using Project contents (optional)

    3. When Use default is enabled, project files are stored in the default workspace directory for projects. If you disable Use default, click the Browse button to select where project files should be stored.

  3. Click the Next > button

    4. The Templates page appears.
