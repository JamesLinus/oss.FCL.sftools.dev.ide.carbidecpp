UIQ UI Design Properties Page

Use the UIQ UI Design Properties page of the New Project Wizard to set the base name and container type for your UI design.

Figure 1 - Setting the Base Name and Container Type

Setting Name and Type

  1. Enter a unique Base Name for your UI Design or accept the default value.
  2. Select the configuration mode for the user interface.
  3. Select the container type from the context menu or accept the default value.
  4. Select the UI Design layout manager.
  5. Click Next

    6. The Basic Settings page appears.
