Layout Configurations

This section allows the user to add, rename, duplicate and remove layouts for UI Design projects.

Figure 1 - The "Layout Configuration" section

Using the Layout Configuration Section

  • Add Button: Used to add a Layout Configuration to the UI Design. If the UI Design is a View, a new View Layout  instance is created. If the UI Design is a Simple Dialog, a CQikContainer instance is created.

The <Add..> button opens a new system window : Add Layout Configuration.
In this page, users are able to edit the properties of the new View Layout. A new View Layout will automatically create a View Page and a Container; these can be configured in the same window.

General Properties:

  • Name: A view layout name can be specified here.

  • Command List: Users are able to select one of the command lists available in the project. If a command list is selected for this property, this will apply for all pages in the project without one command list defined.

View Page Properties:

  • Tab Caption: Allows you to edit the title that will appear in the new tab page.

  • Tab Image: <Edit..> button: Allows you to add an image on the tab of the new view page.

  • Command List: Users are able to select one of the command lists available in the project for the view page component; if this property is selected it will affect only the new page.

CQikContainer

  • Scrollable: Two options are available - Non Scrollable or Scrollable - this option allows  or denies the ability of generating a scroll bar in the container.

  • Layout Manager: Row Layout Manager property is available.

 

Figure 2 - "Layout Configuration" section

  • Rename Button: Used to change the name property of the selected Layout Configuration. This button is enabled only when a Layout Configuration is selected.

  • Duplicate Button: Used to create a copy of the selected Layout Configuration instance. This button is enabled only when a Layout Configuration is selected.

  • Remove Button: Used to delete the selected Layout Configuration instance. This button is enabled only when a Layout Configuration is selected, but disabled when no layout configuration is selected and also when there is only one layout configuration.
