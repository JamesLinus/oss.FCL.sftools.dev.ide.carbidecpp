UI Configurations

This section allows user to add, duplicate and remove UI Configurations for UI Design projects.

Figure 1 - The "UI Configuration" section

 

Using the View Configuration Section

  • List: Displays the UI Configuration instances of the UI Design. It supports single row selection.

  • Add Button: Used to add an UI Configuration to the existing design. If the UI Design is a view, a new ViewConfiguration instance is created. If the UI Design is a SimpleDialog, a new DialogConfiguration instance is created.

  • Duplicate Button: Used to create a copy of the selected UI Configuration instance. This button is enabled only when a UI Configuration is selected.

  • Remove Button: Used to delete the selected UI Configuration instance. This button is enabled only when a UI Configuration is selected.

  • Columns Table:  For Layout Config column and Command Lists column, user is able to change values from default to any other available option.
