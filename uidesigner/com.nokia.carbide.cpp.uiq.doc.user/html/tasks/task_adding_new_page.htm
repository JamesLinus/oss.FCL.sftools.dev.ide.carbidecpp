Adding a New Page Into Existing Project

The main objectives of this page are:

  1.  To provide a way to add a new page for multipage projects or view dialogs.
  2.  To provide a way to configure the properties of the new page.

 

Using "Add View Page"

1. This option is available when users are working with Multipage projects or View Dialogs. To use, select the "View Layout" component from Design or Outline view and invoke the context menu and select the option "Add view Page " as shown in Figure 1.

Figure 1 - The "Add View Page" option

2. After selecting the proper option (Add View Page) from the context menu, a new "Add View Page" window appears (Figure 2).

Figure 2 - The "Add View Page" window

      View Page Properties:

  • Tab Caption: allows you to edit the title that will appear in the new tab page

  • Tab Image - <Edit..> button: allows you to add an image on the tab of the new view page

  • Command List: users are able to select one of the command lists available in the project for the view page component or none

      CQikContainer

  • Scrollable: two options are available - Non Scrollable or Scrollable - this option allows  or denies the ability of generating a scroll bar in the container.

  • Layout Manager: Row Layout Manager property is available.

3. After pressing <OK> button, a new page will be added in the design with properties as configured.
