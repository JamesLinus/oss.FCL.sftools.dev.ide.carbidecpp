Selecting Application Editor Pages

The Application Editor contains multiple pages. Use the tabs at the bottom of the editor to select the desired page.

Figure 1 - Application Editor Overview Page

Selecting a Page

  1. Open the Application Editor (or select the associated tab if the editor is already open).

    2. The application editor is displayed. Four tabs along the bottom of the display allow you to select a specific editor page.

    NOTE When you first open the Application Editor, the Overview page is shown.

  2. Click the tab representing the page to open.

    3. The desired page of the Application Editor will be displayed.
