Screen Layout Configurations

The latest SDK’s for the UIQ 3.x platform support multiple screen resolutions:

Table 1. UIQ style - screen mode - display size

Style Screen Mode Display Size Touche Screen Interaction Style
Pen Style  Portrait 240x320 Yes Menu Bar
Pen Style Landscape Landscape 320x240 Yes Menu Bar
Softkey Style Portrait 240x320 No Softkeys
Softkey Style Small Portrait, small 240x256 No Softkeys
Softkey Style Touch Portrait 240x320 Yes Softkeys

NOTE Availability of screen layouts is dependent upon the SDK used by the project. Not all SDK's support all layouts.
