UIQ 3.x UI

UIQ 3.x software development kits (SDK) enable application development for Symbian OS phones featuring the UIQ Platform. UIQ allows mobile phone developers to create applications for a highly diversified range of phones on one code line, supporting:

  • Several Phone Styles
  • Enhanced Customization Options
  • Richer User Experience

UIQ devices build on the features provided by the Symbian OS.
