AppUi Key Bindings

The Application Editor AppUi page supports these key bindings for selecting and nudging components, as well as scrolling the view.

Table 1. Application Editor AppUi page—key bindings
Action Key Binding Description
Selection
Up arrow
Selects the next sibling physically located above the current selection.
Down arrow
Selects the next sibling physically located below the current selection.
Left arrow
Selects the next sibling physically located to the left of the current selection.
Right arrow
Selects the next sibling physically located to the right of the current selection.
ALT up arrow
Select the parent container of the current selection.
ALT down arrow
Select the first child of the currently selected container.
CTRL arrow (up, down, left, right)
Specify potential parent/child in multiple selection with current selection -
Space bar processes selection
SHIFT arrow
Multiply select next sibling with current selection (no confirmation needed)
SHIFT-ALT arrow
Multiply select parent/first child with current selection (no confirmation needed)
Nudging
F5
 Nudge current selection up.
F6
 Nudge current selection down.
F7
 Nudge current selection left.
F8
 Nudge current selection right.
ALT-F5
 Nudge current selection up about 10 pixels at 100% magnification.
ALT-F6
 Nudge current selection down about 10 pixels at 100% magnification.
ALT-F7
 Nudge current selection left about 10 pixels at 100% magnification.
ALT-F8
 Nudge current selection right about 10 pixels at 100% magnification.
Scrolling
CTRL-SHIFT arrow
Scroll editor viewer if it's not already fully visible.
