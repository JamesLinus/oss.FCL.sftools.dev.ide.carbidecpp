Languages Page

The Application Editor Languages page allows you to add support for additional languages to your project and remove support for current languages. Use it to specify which language is displayed in the UI Designer. It also allows you to specify the source code format for localizable strings.

Figure 1 - The Application Editor Languages page

Table 1. Application Editor Languages page—items
Item Explanation
Included languages Shows which languages are supported in the application. Click Add to add more languages to the project. Click Remove to remove a language from the project. UI Designer automatically updates its generated files to reflect the changes.
Localization format Choose the format that generated source files use for localized strings.
