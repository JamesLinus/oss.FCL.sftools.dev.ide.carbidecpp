UIQ UI Design Properties Page

Use the UIQ UI Design Properties page of the UI Design wizard to specify a base name, the UI configuration mode (Softkey portrait style, Softkey touch portrait style, Pen touch portrait style), container type (Non Scrollable, Scrollable) and layout manager (Row Layout Manager, Grid Layout Manager)  for your UI Design.

Figure 1 - UIQ UI Design Properties page of the UI Design Wizard

The UI Design Wizard pages in sequential order:
  • Application Properties Page
  • UIQ UI Design Selection Page
  • Current: UIQ UI Design Properties Page
