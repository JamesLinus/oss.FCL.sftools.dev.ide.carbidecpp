Layout Components

Layout defines compound components. These components can be compounded of any other component from the palette.

The Layout  components include:

  • Container - Is the main compound component because it is the base where all the other components will be placed. A Layout Manager component needs to be created as default any time a container is created. Containers can contain containers as children also (Nested containers).
  • Row Layout Manager - Default row layout manager for container.
  • System Building Block - A building block is a compound control that has the task of laying out its component controls.
