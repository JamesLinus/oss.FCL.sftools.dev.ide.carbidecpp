Controls Components

The Controls components include:

  • Bitmap Button - A button using an image
  • Choice List - Drop-down list of text items
  • Choice List Item - This is one item in a choice list
  • Color Selector - Allows selecting a color from a pop out palette
  • Combo Box - Control providing a drop-down  list and an editor
  • Command Button - A standard button with a label and an image
  • Image - An image widget
  • Label - Component that holds and displays a text
  • Labeled Check Box - A list of labels with an associate check box
  • List Box  -  Implements the basics of a list box.
  • Progress Info - This component dynamically shows job progress.
  • Text Button - A button with two text labels
  • Vertical Option Button List - Set of vertical elements working as buttons
