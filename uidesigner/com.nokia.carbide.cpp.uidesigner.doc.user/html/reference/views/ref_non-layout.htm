Non-layout Pane

Use the non-layout pane of the UI Design Editor for the objects of your UI Design which are not always visible when the associated screen is being displayed, such as a screen's associated menus and dialogs.

Drag the desired component type in the palette to the non-layout pane, or select the desired type from the component palette and then click within the non-layout pane to add a non-layout object of that component type to your design. The cursor will display a plus sign (+) when it is in an area where you may add a component of the selected type to your UI Design.

Select objects within the non-layout pane to examine, modify, or delete them. If a selected non-layout component has a visible portion, it will appear in the layout region of the view. For example, selecting a menu component displays the menu editor in the layout region, enabling you to modify the menu properties directly.
