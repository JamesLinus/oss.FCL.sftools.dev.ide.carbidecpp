Bitmap Editor

The UI Designer enables you to edit bitmaps using the Windows Bitmap Editor directly within the toll environment. The Bitmap Editor supports all of the familiar graphic editing conventions and tools found in any basic graphics editing program.

NOTE Integration of this OS-provided feature within the Carbide.c++ environment is not complete. There is no facility provided to create files or save existing files with a new name.
