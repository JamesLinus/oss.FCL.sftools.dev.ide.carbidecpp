Properties View

Use the Properties view to examine and modify the parameters of the objects in your UI Designs.

When an object is selected, its defining characteristics are listed by name and value in the Properties view. Properties have types with corresponding cell editors for enumerated types, boolean values, items from predefined lists, dates, co-ordinates, numbers, text, and any others. Some editors provide a control () in the right-hand side of the Value column which open special purpose editors for components like images, arrays, and string lists.

Use the tree controls to access subfields of a structure or elements of an array property, whose subproperty names are the element’s array index.
