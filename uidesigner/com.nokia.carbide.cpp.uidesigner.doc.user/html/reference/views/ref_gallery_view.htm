Gallery View

Use the Gallery view to see a visual representation of all UI Designs in all projects in your current workspace. Once open, use the gallery context menu to open a selected UI Design or change the magnification of the UI Designs within the view. The Gallery view automatically updates itself whenever a design changes so that it always displays the current design. Select Window > Show View > Other... and select Gallery in the Carbide.c++ folder to display the Gallery view.

Use the disclosure triangles (see red box in Figure 1) to display/hide the UI Designs for each project in the worksapce.
