Palette Settings

Use the Palette Settings dialog to specify the appearance and behavior of the Palette view.

Figure 1. The Palette Settings dialog

Table 1. Palette Settings dialog—items
Item Explanation
Font Click Change to choose a font other than the Workbench Dialog Font (default) or Restore Default to return to the original setting.
Layout Specify how the Palette view displays components. The layout options include:
Layout option Example
Columns
List (default)
Icons only
Details

NOTE You can also right-click the Palette view and choose the layout type from the Layout item on the context menu.

Use large icons Checkmark to display larger icons in the Palette view no matter which Layout option is chosen.

NOTE You can also right-click the Palette view and set this option from the context menu.
Drawer Options Controls the behavior of the Palette drawer.

Opening the Palette Settings Dialog

  • Right-click in the Palette view and choose Settings... from the context menu

    • The Palette Settings dialog appears.
