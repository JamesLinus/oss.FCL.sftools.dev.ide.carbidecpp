Toolbar

Use the UI Design Editor toolbar to perform basic tasks on selected objects in your UI design.

Figure 1 - The UI Design Editor's toolbar

Table 1. UI Designer toolbar options
Commands Icons Deescription UI Design
Editing
Standard Undo, Redo, Cut, Copy, and Paste operations. s60, UIQ
Aligns Objects
Align selected objects vertically or horizontally. S60 only
Matching Objects' Height or Width
Match width and height of selected objects. S60 only
Setting UI Design Magnifications
Control the magnification of the layout area. s60, UIQ
Palette Filtering

Enable palette filtering to dynamically update the Palette to only show components that can be placed into the UI design and hiding all unusable components. When disabled, all components are shown.

Palette filtering is enabled by default.

s60, UIQ
Setting Screen Layout Configurations
View your UI design using any of the standard S60 and UIQ layout configurations. s60, UIQ
