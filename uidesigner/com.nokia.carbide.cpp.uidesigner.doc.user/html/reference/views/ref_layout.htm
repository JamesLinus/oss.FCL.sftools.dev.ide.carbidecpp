Layout Pane

Use the layout pane of the UI Design Editor to build the screens for your application and edit their objects visually.

Drag from the desired component type in the palette to the layout pane, or select the desired type from the component palette and then click within the layout pane to add a layout object of that component type to your design. The cursor will display a plus sign (+) when it is in an area where you may add a component of the selected type to your UI Design.

Select objects within the layout pane to examine, modify, move, resize, align, or delete them.

NOTE The UI Designer will enforce S60 UI Design conventions for sizing, spacing and relative positioning of objects which you place in most of the top level container objects. Objects placed in an "Empty" container may be moved and sized without regard to their absolute and relative placement within the container.
