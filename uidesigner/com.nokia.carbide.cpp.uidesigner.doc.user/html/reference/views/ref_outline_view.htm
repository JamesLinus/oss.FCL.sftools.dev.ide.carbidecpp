Outline View

Use the Outline view to examine the hierarchical structure of the objects in your UI designs. You can use the tree controls to hide or display the child objects of a higher level object. Each object in the UI design is identified by icon, a unique name (either automatically generated or modified by you) and the component type.

You can select one or more objects in the Outline view. When an object is selected, its defining characteristics are listed by name and value in the Properties view. You can also reorder or delete objects from the view.
