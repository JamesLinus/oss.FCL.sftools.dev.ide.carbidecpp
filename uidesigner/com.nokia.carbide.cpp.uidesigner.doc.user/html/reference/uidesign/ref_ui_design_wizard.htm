UI Design Wizard

The UI Design Wizard leads you through each of the steps necessary to configure a UI Design for use with the Carbide.c++ UI Designer. The wizard consists of a series of pages, each of which collects information of a specified type. When you complete all of the pages, Carbide.c++ will create a new UI Design with the settings you have specified.

Figure 1 - New UI Design wizard

Launching the UI Design Wizard

  • Select the File > New > S60 Ui Design menu item

    • The New S60 UI Design wizard appears.
