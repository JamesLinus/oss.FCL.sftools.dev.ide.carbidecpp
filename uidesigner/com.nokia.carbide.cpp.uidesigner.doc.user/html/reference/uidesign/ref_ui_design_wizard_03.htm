Content Variants Page

Use the Content Variants page of the UI Design Wizard to provide the additional required information if required by the chosen base container like a List Box. It is displayed automatically if you select an initial container type of List Box. The List Box container type requires additional information before it can be fully defined.

Figure 1 - The Content Variants Page of the UI Design Wizard

The UI Design Wizard Pages in Sequential Order:
  • Application Properties Page
  • S60 UI Design Selection Page
  • Current: Content Variants Page
  • Container Name and Type Page
