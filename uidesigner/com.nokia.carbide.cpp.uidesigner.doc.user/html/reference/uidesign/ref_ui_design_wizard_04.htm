Container Name and Type Page

Use the Container Name and Type page of the UI Design wizard to specify a name and container type for your UI Design, and to specify whether to implement automatic Avkon view switching for this design.

Figure 1 - The Container Name and Type page of the UI Design Wizard

The UI Design Wizard Pages in Sequential Order:
  • Application Properties Page
  • S60 UI Design Selection Page
  • Content Variants Page
  • Current: Container Name and Type Page
