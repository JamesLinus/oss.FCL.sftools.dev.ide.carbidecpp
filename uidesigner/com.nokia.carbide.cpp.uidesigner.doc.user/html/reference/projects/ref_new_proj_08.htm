Basic Settings Page

The New Project Wizard leads you through each of the steps necessary to configure a project for use with the Carbide.c++ UI Designer. The wizard consists of a series of pagess, each of which collects information of a specified type. When you complete all of the pages, Carbide.c++ will create a new project using the settings you have specified.

Use the Basic Settings Page of the New Project Wizard to specify a unique Application UID and to set the author and copyright notice for your application.

NOTE For most uses, the value of the Application UID provided by Carbide.c++ upon the initial display of this page will be an acceptable default.
