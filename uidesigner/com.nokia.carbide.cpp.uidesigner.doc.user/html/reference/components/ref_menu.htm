Menu Components

The Menu components include:

  • Menu Bar - a standard Eikon menu bar
  • Menu Item - a standrd Eikon menu item
  • Menu Pane - a standard Eikon menu pane
  • System Markable List Menu Pane - a standard menu pane for the mark, unmark, mark all, and unmark all menu commands
  • System Menu Pane - a menu pane that comes from an existing resource
