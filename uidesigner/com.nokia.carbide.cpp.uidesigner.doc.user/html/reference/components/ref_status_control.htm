Status and Control Components

Status and Control components are a collection of components that provide status feedback and common controls used in applications.

The Status and Control components include:

  • Context Icon - the icon appearing in the status pane
  • Control Pane - an S60 Control Button Class array
  • Navi Pane Image - an image appearing in the Navigation pane of the status area
  • Navi Pane text - a text label appearing in the Navigation pane of the status area
  • Status Pane - the standard S60 component displaying the title pane, context icon, navi pane, and system indicators
  • Title - the title appearing in the status pane
