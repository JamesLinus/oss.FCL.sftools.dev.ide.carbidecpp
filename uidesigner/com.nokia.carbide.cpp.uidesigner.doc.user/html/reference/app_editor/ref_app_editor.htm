Application Editor

Use the Application Editor to set application-level settings for the UI of your S60 application. The Application Editor is comprised of four information pages that include:

  • Overview - a project summary page
  • UI Designs - manages the UI designs present in the project
  • Languages - manages the languages used by the project
  • AppUi - design the program interface

Figure 1 - Application Editor Overview page

Opening the Application Editor

  • Double-click the application.uidesign file in the Project Explorer view

    • or

  • Right-click the application.uidesign file and choose Open

    • The Application Editor Overview page appears.
