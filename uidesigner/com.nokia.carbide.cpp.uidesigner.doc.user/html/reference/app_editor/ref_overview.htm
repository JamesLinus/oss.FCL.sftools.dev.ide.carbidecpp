Overview Page

The Application Editor Overview page has only a single user adjustable setting. It allows you to specify the baseline SDK which will be used to generate source code and select components.

Figure 1 - The Application Editor Overview page

Table 1. Application Editor UI Designs page—items
Item Explanation
Summary A brief statement on the project status.
Baseline SDK

Select from the dropdown list which SDK the project should use as its baseline for component compatibility.

NOTE Options available for this setting are limited to SDKs which are no later than the earliest build SDK which you selected when initially creating the application project.
