UI Designs Page

The Application Editor UI Designs page allows you to remove existing UI Designs, import UI Designs, and change the order of the UI Designs within your application. It provides an alternate method to add a new UI Design to your project. It also provides a way for you to indicate that you wish to enable automatic tabbing between some Avkon views and to specify the text or image to display in the tab.

Figure 1 - The Application Editor UI Designs page

Table 1. Application Editor UI Designs page—items
Item Explanation
Remove Removes the currently selected UI design from the All UI Designs list. The UI design is not removed from the project, only references to it. Use Import to add the UI design back into the UI Design list.
Move Up / Move Down Moves the currently selected UI design up or down in the UI Design list. If Enable Navigation Pane Tabbing is enabled, this will reorder how the UI designs appear in the tabs.
Import Click Import to add a previously removed UI design back into the UI Design list.
Enable Navigation Pane Tabbing Enable to automatically configure the application to support navigation pane tabbing.
Is in ApUi tab group Enable to activate the Tab Details section for tab customization.
Text Enable to add a text title to the tab, then enter the title into the text field provided.
Image Enable to add an image to the tab. Click Edit to open the Edit Image Property dialog to select an image.
