Add Symbian Languages Dialog

The Application Editor Languages page allows you to add support for additional languages to your project. Use the Add... button on the Application Editor Languages page to display the Add Symbian Languages Dialog.

The Add Symbian Languages Dialog allows you to add one of more supported languages to your UI Design.
