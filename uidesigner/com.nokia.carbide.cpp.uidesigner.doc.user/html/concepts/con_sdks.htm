SDKs

SDKs are based on a particular UI platform for Symbian OS. The UI Designer supports multiple Symbian-based SDKs. A single SDK is associated with a single project. A project can be converted to another SDK. The baseline SDK you are using will provide default components. The component toolbox shows the available components supported by the SDK.
