Standard Components

The UI Designer provides components for the most commonly used interface elements. Interface components are visible in the Component Palette when using the UI Design Editor or the AppUi pages. Components in the Palette are placed in a design using one of two methods, including:

  • Select and drop — click a component in the Palette view to select it, then move the pointer over the design's layout or non-layout regions. The cursor changes shape indicating the component can either be dropped in the design or not. In addition, a tool tip appears providing helpful information on the component. Click in the design to add the component.
  • Drag and drop — click a component in the Palette view and drag it into the design. Release the mouse button to add the component to the design. No tool tip appears when using this method of adding components to a design.

Standard components include:

  • Control Components
  • Editors Components
  • Menu Components
  • Misc Components
  • Notes and Dialog Components
  • Status and Control Components
