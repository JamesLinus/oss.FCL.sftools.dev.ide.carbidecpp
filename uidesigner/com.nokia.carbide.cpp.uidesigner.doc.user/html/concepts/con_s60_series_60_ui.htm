S60 UI

Nokia S60 software development kits (SDK) enable application development for Symbian OS phones featuring the S60 Platform. Nokia S60 UI is intended for use in higher end mobile phones featuring personal information management (PIM) and multimedia applications. S60 devices build on the features provided by the Symbian OS.
