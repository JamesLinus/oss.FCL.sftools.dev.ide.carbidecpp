Reordering Objects

You can change the order of objects within your UI Design's containers within the Outline view. You may also change the order of items within menus.

NOTE Objects which are displayed in the non-layout region have no intrinsic order and thus cannot be re-ordered.

Reordering Objects

  1. Open a UI Design in the UI Design Editor or select an open UI Design.

    2. The selected UI Design will be displayed.

  2. If the Outline view is not currently displayed, issue the Window > Show View > Outline command.

    3. The Outline view will be displayed.

  3. Click and hold the object which you wish to reorder.

    4. The selected object will display a greyed background, as shown in Figure 1.

    Figure 1 - A selected object in the Outline view

  4. Drag the object to the desired location within the area which contains the other sub-objects of the selected object's parent container.

    5. The cusror will indicate when you are at an ineligible reordering location, as shown in Figures 2.

    Figure 2 - The cursor showing an ineligible reorder destination

    When you are at an eligible reordering location, a dark black line will appear as shown in Figure 3.

    Figure 3 - The cursor showing an valid reorder destination

  5. Release the mouse button.

    6. The object's order within the container will be updated.

NOTE Reordering of objects is Undoable
