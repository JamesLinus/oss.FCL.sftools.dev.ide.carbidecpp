Adding New Objects

You add objects to your existing UI designs visible in the UI Design Editor view using one of the following methods:

  • Select the component, then click in the design itself to place it in the design
  • Click and drag the component to the design

The component is added to the UI design (layout or non-layout regions as appropriate) and also appears in the Layout view.

Figure 1 - Ready to add a Menu Bar object to the non-layout area

NOTE The UI Design Editor enforces many S60 design principles. It will not permit you to add an object to a container which does not accept objects of that type.
