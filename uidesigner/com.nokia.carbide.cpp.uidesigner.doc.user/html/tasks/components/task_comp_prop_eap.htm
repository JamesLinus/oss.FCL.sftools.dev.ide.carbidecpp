Inspecting/Modifying Array Properties

Use the Edit Array Property dialog box to add or remove elements of an array associated with items property of Enumerated Text Popup Setting Item controls. The Binary Popup Settings Item also includes an items property its elements are fixed and the property comes prepopulated with the items. In either case the subproperties of the items can be edited using this dialog. Another property that is edited using this editor is the additional images property of a list box.

Inspecting Array Properties

  1. Open a Properties view for a component that contains an items property

    2. The Enumerated Text Popup Setting Item control, the Binary Popup Setting Item control, and the additional images property of a list box include the items property.

    Figure 1. Open the Edit Array Property dialog box

  2. Click the browse control

    3. The Edit Array Properties window appears (Figure 2).

    Figure 2. Edit Array Property dialog box

  3. Edit the array and array element properties

    4. If the component is a Binary Popup Setting Item control the item properties are prepopulated and cannot be redefined. For other components you must add elements, define their settings, and reorder them as desired.

    In the Edit Array Property dialog box (Figure 2) you can add (or remove predefined) array elements for text popup items. Use the up and down arrows to arrange the order of the elements. In the Value column enter a value directly into the line or select the line and click the button that appears in the column to open the String List Editor dialog.

  4. Click OK

    5. The Edit Array Properties window closes and the array elements shown in the Value column.
