Selecting Objects

Before you can work with an existing object it must be selected. The UI Design Editor provides multiple ways to select one or more objects. Some commands, such as alignment commands, are available only when multiple objects are selected.

Selecting Objects

  • In the UI Design Editor view :
    1. Click on an object

      A bounding rectangle appears around the object (Figure 1), the same object is highlighted in the Outline view, and its properties are shown in the Properties view.

    2. To add additional objects to the selection:
      • Shift-click or control-click an object to select it and all objects between the first selected object and the second

        Another bounding rectangle appears around the newly selected object (Figure 1).

        NOTE The properties of the first object selected continue to be shown in the Properties view.

         

        Figure 1 - Extending selections in the UI Design view

      The selected object(s) can than be moved, resized, aligned, or deleted as required.

  • In the Outline view:
    1. Click on an object

      The object is highlighted in the list (Figure 2) and the UI Design Editor view, and the object properties are shown in the Properties view.

    2. To add additional objects to the selection:
      • Shift-click an object to select it and all objects between the first selected object and the second (Figure 2)
      • Control-click an object to perform a discontinuous selection (Figure 2)

        Another bounding rectangle appears around the newly selected object (Figure 1).

        NOTE The properties of the first object selected continue to be shown in the Properties view.

       

      Figure 2 - Extending selections in the Outline view

      The selected object(s) can than be moved, resized, aligned, or deleted as required.

      NOTE Shift-clicking on any object will add it to the selection, but shift-clicking will not remove an object once it is selected.

      NOTE Control-clicking on any object will toggle its inclusion in the selection.
