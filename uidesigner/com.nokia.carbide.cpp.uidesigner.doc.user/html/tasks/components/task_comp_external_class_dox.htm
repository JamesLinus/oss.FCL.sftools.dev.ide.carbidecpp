Viewing Class/Resource Documentation

If your installation includes addressable SDK help files, the Palette's Container Information pane will implement links to the documentation for components.

Figure 1 - The Component Information pane

Viewing Class/Resource Documentation

  1. Select a component type in the Palette.

    2. The Component Information pane will display the Class name and Resource type for the selected component.

    NOTE The Component Information pane's label will display Select when no component is selected.

  2. Click the linkable data in the Component Information pane to view the associated documentation

    3. The documentation will be displayed.

    NOTE If you do not have addressable SDK help files installed, the Class name and Resource type will display as plain text instead of as links.
