Displaying the Outline View

Use the Outline view to see all the objects in your UI Design.

Figure 1 - The Outline View

NOTE The Outline view will be displayed empty when no UI Design is open.

Displaying the Outline View

  1. Click Window > Show View > Outline

    2. The Outline view will be displayed as shown in Figure 1. If a UI Design is open, its objects will be displayed in the Outline view .

  2. Use the tree controls to display or hide subsets of the objects in the UI Design.
