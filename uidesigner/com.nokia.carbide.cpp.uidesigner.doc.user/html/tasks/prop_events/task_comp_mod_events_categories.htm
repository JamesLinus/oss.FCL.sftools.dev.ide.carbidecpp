Showing/Hiding Event Categories

The Events view lists each supported event type for each object within your UI Designs and indicates the name of the asociated handler for those events you have chosen to implement.

The Events view can display the list of supported events for each object by event type or as one inclusive list as shown in Figure 1.

 

Figure 1 - Events by category and list

Showing/Hiding Event Categories

  1. Select an object in a UI Design.

    2. The object will display a bounding rectangle with squares at its corners and midpoints.

  2. Display the Events view.

    3. The event list for that component type will be displayed.

  3. Right-click in the content area of the Events view.

    4. The Events context menu appears.

    NOTE The context menu will contain a checkable entry titled Show Categories. Its checked status indicates whether the Events view is currently displaying events by category.

  4. Check or uncheck the item as desired.

    5. The Events list will be redisplayed as indicated.
