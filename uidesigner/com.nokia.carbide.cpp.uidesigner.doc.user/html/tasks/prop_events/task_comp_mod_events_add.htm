Adding a New Event Handler

The UI Designer will implement shell files for each event you wish to handle within your application. It is up to you how to implement the details of how to handle each event.

The Events view provides a quick way to add event handler code for any supported event.

Adding a New Event Handler

  1. Select an object which responds to events.

    2. The object is displayed with the selection rectangle .

  2. Display the Events view (if not already visible) using Window > Show View > Events.

    3. The Events view displays all events associated with the selected object type. Events for which a handler already exists display the name of the handler next to the event.

  3. Double-click in the Function column for the event you wish to handle to generate and jump to the source code which implements the handler .

    4. NOTE You may be asked to confirm saving modified information.

    The associated source file is opened to the new handler routine for the indicated event.
