Assigning Project Directories

Use the Project Directories page of the New Project Wizard to accept the default names, or provide custom names for the directories used within your project.

Figure 1 - Assigning project directory names

NOTE To avoid errors, use default directory names whenever possible.

Assigning Project Directory Names

  1. You may accept the default directory names, or enter custom names for any or all of the directories displayed in Figure 1.
    • Include
    • Source
    • Group
    • Data
    • Install
    • Graphics
  2. When you are satisfied with the names for all directories, click the Finish button to complete the wizard.

    3. The wizard creates a new project and displays it in the Project Explorer view.
