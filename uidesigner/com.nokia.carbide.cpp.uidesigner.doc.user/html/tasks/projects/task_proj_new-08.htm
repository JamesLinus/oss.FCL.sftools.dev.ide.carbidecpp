Entering the Application Basic Settings

Use the Basic Settings Page of the New Project Wizard to set the Application UID, Author, and Copyright notice for your project.

Figure 1 - Entering the Application Basic Settings

Entering the Application Basic Settings

  1. Enter the Application UID.

    2. For most cases, you may use the default value which is provided by the UI Designer.

  2. Enter the Author.
  3. Enter the Copyright notice.
  4. Click the Next > button.

    5. The Project Directories Page will appear.

    NOTE If you do not need to provide custom names for the directories within your project, you can click the Finish button on this page to complete the wizard. Upon completion, the wizard will display your new project in the Project Explorer view.
