Creating New GUI Projects for the UI Designer

The New Project wizard will guide you through the steps needed to create a new GUI project.

This page provides a list of the steps needed to create a new S60 C++ GUI project using the wizard. The links in the discussion of each step take you to a page which contains a detailed description of the subtasks needed to perform that step.

Some common New Project Wizard actions include:

  • Click the Cancel button to terminate the wizard without creating a project
  • Click the Back button to return to the previous page of the wizard
  • Click the Next button to accept all entries on the current page and continue to the next page of the wizard.
  • Click the Finish button (when it is enabled) to complete the wizard.

Creating New S60 C++ GUI Projects

  1. Start the New Project Wizard

    2. Select the C++ Application for S60 Project wizard from the File > New submenu.

    The wizard will guide you through the tasks needed to create a new S60 C++ GUI project.

  2. Select the project's template

    3. Select the template to use when creating the project in the Templates page.

  3. Set the project name

    4. Specify the project name and the location to store project contents in the Symbian OS C++ Project page.

  4. Select the project SDK's

    5. Assign one or more SDKs to the project in the Symbian OS SDKs page.

  5. Set basic application properties

    6. Set the application name, baseline SDK, and language in the Application Properties page.

  6. Select the UI Design initial content

    7. Select the initial content type for your UI Design in the S60 UI Design Selection page.

  7. Select the UI Design content variant (optional based on choice for initial content)

    8. Select the desired variant of the initial content type in the Content Variants page.

  8. Set the base name and container type

    9. Set the base name and container type for the initial content in the Container Name and Type page.

  9. Enter the Application UID

    10. Fill in a unique Application UID in the Basic Settings page. In most cases you may use the default value which is provided.

  10. Assign project directories

    11. Fill in the project directory names to use when creating the project and copying files in the Project Directories page.

    NOTE The default directory names should be used except where local software development practices specify other directory names.

Once all of these steps are complete, a new Symbian OS project is created on disk and displayed in the Project Explorer view.
