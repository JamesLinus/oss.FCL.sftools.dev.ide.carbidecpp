Selecting the Content Variant

Use the Content Variants page of the New Project Wizard to select the desired variant of your initial content. For example, the List Box container type requires additional information before it can be fully defined for initialization by the UI Designer. It will be displayed automatically if you select an initial container type of List Box.

Figure 1 - Selecting the Content Variant

NOTE This screen only appears if your intial content type choice was a List Box.

Selecting the Content Variant

  1. Select the content variant by clicking on its picture.

    2. NOTE There are more variants available for this type than can be displayed simultaneously. Use the scroll control to view the remaining variant options.

  2. Click the Next > button.

    3. The Container Name and Type Page will appear.
