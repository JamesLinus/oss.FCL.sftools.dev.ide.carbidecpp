Selecting the Display Language

The Languages page of the Application Editor displays a list of all languages which are currently enabled for your application and indicates which language will be displayed in the UI Designer.

Use this page to change the Ui Designer display language.

Selecting the Display Language

  1. Select the Languages page of the Application Editor.

    The Languages page ise displayed.

    Figure 1 shows the display for an application which supports four languages.

    American English is currently selected for display in the UI Designer, as indicated by the associated check box.

    Figure 1 - The "Included languages" pane of the Languages page.

  2. Check the box associated with the language which you wish to display in the UI Designer.
  3. Close the application.uidesign file.

    The Save Resources confirmation dialog will appear.

  4. Click the Yes button to confirm saving the change of display language.

    Any UI Designs which you open will display the newly selected language.

    NOTE If you have any UI Designs open when you complete this step, they will update their display to show the newly selected display language.
