Adding a Language

The Languages page of the Application Editor displays a list of all languages which are currently enabled for your application and indicates which language will be displayed in the UI Designer.

Use the controls on this page to add support for additional languages.

Adding Support for Additional Language(s)

  1. Select the Languages page of the Application Editor

    2. The Languages page is displayed.

    Figure 1 shows the display for an application which supports three languages.

    Figure 1 - The "Included languages" pane of the Languages page.

  2. Click the Add... button

    3. The Add Symbian Languages dialog appears, as shown in Figure 2.

    Figure 2 - The Add Symbian Languages dialog adding French and Spanish.

    NOTE The listing does not include languages already supported by your application.

  3. Check the box(es) for the language(s) you wish to add
  4. Click the OK button

    5. The Included languages list on the Languages page is updated to include the additions.

  5. Close the application.uidesign file

    6. The Save Application Design confirmation dialog will appear.

  6. Click the Save button to confirm saving the addition of the language(s) to the application.uidesign file

    7. The UI Designer creates (or updates) the localization files of the appropriate type for the current localization format.

    NOTE The text labels in the generated localization files are initialized to "??" + <the text of the currently selected display language> to indicate the lines that require translation.
