Changing the Baseline SDK

Use the Application Editor’s Overview page to set the project’s current Baseline SDK. The Baseline SDK specifies the minimum SDK version UI Designer uses to generate source code and display available components.

Figure 1 - The Overview Page of the Application Editor.

Selecting a different baseline SDK

  1. Open the Application Editor, or switch to the Overview Page if it is already open.

    2. The Overview Page will be displayed.

  2. Click the context menu displayed in the Baseline SDK pane.

    3. The context menu will list the appropriate subset of your installed SDKs.

    NOTE The SDKs listed will be limited to those which are no later than the earliest build SDK.

  3. Click the desired SDK in the context menu.

    4. The menu will display the selected SDK as the project baseline.
