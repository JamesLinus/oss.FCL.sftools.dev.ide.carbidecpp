Importing a UI Design

When you create a UI design with the UI Designer it is automatically included in the builds of the project to which you originally assign it. If you decide that you do not want that UI Design to be part of your project, you may permanently delete it or temporarily remove it from the project.

Later, if you wish to restore a UI Design which you have temporarily removed - or if you wish to copy a UI design originally created for a different project into your project - you must import the UI design into the project for it to be included with subsequent builds.

Importing a UI Design

  1. Open the Application Editors UI Design page

    2. NOTE If the Application Editor is already open, bring it to the front by clicking on its tab.

  2. Click Import.

    3. The Import UI Designs dialog is displayed as shown in Figure 1.

    Figure 1 -The Import UI Designs dialog.

    This dialog will display all UI Designs which are available for import. It will include any which you have previously removed from the project as well as any which you have properly copied into the project.

  3. Select the UI Design(s) to import.

    4. You may select individual UI Designs by clicking their associated check boxes. All checked UI designs will be imported while those left unchecked will not.

    You can also click Select All to select all of the UI designs available for importing, or click Deselect All to unselect all UI designs currently selected.

  4. Click OK

    5. The Import UI Designs dialog is closed and the imported UI Designs list shows the newly imported UI designs. If these designs are part of the application's navigation tab group, you may wish to reorder them.
