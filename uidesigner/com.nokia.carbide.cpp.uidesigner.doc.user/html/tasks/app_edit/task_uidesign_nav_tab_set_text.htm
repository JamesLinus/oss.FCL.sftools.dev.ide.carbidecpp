Setting the Tab Text

After you have enabled a UI Design for inclusion in the tab group you can specify the text to display in the tab for that view.

Setting the Tab Text

  1. Open the Application Editors UI Design page to view the UI Design Details section

    2. NOTE If the Application Editor is already open, bring it to the front by clicking on its tab.

    Figure 1 - UI Design Details in the UI Designs page

    The All UI section of the UI Designs page contains a list of all UI Designs currently included in the project.

  2. Select the UI Design to set the tab text

    3. The UI Design is highlighted to indicate that it has been selected.

  3. Checkmark the Enable Navigation Pane Tabbing option

    4. If this box was not already checked, the UI Design Details controls will be displayed on the right side of the UI Designs page.

    NOTE You must check this box before you can add any of your UI Designs to the tab group.

  4. Checkmark the Is in AppUi tab group option to add the selected design to the tab group

    5. When the box is checked, exactly one of the radio buttons below it will be selected.

  5. Checkmark the Text button
  6. Enter text into the field to the right of the Text option (Figure 1)
  7. Select the File > Save menu item

    8. NOTE These changes will not become permanent until your next save of the application.uidesign file.
