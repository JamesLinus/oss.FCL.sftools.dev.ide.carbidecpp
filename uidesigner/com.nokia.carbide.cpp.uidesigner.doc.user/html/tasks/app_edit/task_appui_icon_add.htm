Adding an AppUi Context Icon

The AppUi page allows you to set global aspects of your application interface including the AppUi context icon. This icon is displayed in the status pane of all screens in the application, unless over-ridden.

Adding an AppUi Context Icon

  1. Select the AppUi page of the Application Editor.

    2. The AppUi page is displayed.

  2. Open the Status and Control drawer of the palette.

    3. The associated component types are displayed.

    NOTE If Application Icon is not displayed as an available type the project already has an AppUi context icon specified.

  3. Drag the Application Icon into the status pane area of the layout region and release (Figure 1).

    4. An Application Icon is added to the AppUi status pane.

    Figure 1 - Placing the application icon component into the AppUi status pane.

  4. Right-click the Application Icon component and choose Edit Image Property

    5. The Edit Image Properties dialog opens allowing you to set the application icons image.
