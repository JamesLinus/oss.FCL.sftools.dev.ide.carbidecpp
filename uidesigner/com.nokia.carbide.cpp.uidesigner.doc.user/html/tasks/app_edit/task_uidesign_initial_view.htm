Selecting the Initial View

The UI Designs Page of the Application Editor displays a list of all UI Designs within your application.

It allows you to specify the first screen to display, and to add, import, remove, and reorder the UI Designs within your project. It also allows you to enable support for Navigation Pane Tabbing.

Figure 1 - The All UI Designs pane with the initial UI Design indicated

Selecting a different initial screen.

  1. Open the Application Editor, or switch to the Overview Page if it is already open.

    2. The Overview Page will be displayed.

  2. Switch to the UI Designs Page.

    3. The UI Designs Page will be displayed.

  3. Check the box for the UI Design which you wish to be initially displayed.

    4. NOTE This change will not become permanent until your next save of the application.uidesign file.
