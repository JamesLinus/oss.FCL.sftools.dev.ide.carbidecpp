Editing the AppUi Title Pane Text

The AppUi page allows you to set global aspects of your application interface.

NOTE The title pane text is edited just as any other editable text field.

Editing the AppUi Title Pane Text

  1. Select the AppUi page of the Application Editor.

    The AppUi page is displayed.

  2. Edit the title using the direct edit method (Figure 1)

    Figure 1 - Direct editing of AppUi title pane

    1. Double-click in the caption region of the status pane.

      The title caption field is selected.

    2. Type in a new title.

      The new title replaces the old one as you type it.

    3. Click elsewhere to end the edit.

      The new title is displayed.

  3. Or, edit the title pane in the Properties view (Figure 2)

    Figure 2 - Editing of AppUi title pane text in Properties view

    1. Click the Data > long caption or Data > short caption properties

      The chosen properties field is selected.

    2. Type in a new name for the chosen property

      The new title replaces the old one as you type it in the AppUi pane.

    3. Press Enter

      The new title is displayed.
