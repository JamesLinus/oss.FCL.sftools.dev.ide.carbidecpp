Reordering UI Designs

The UI Designs page of the Application Editor displays all UI Designs within your project in the order they will appear in the tab group. You may specify the tab group order for each UI Design by changing its placement in this list.

Reordering UI Designs

  1. Open the Application Editors UI Design page to view the All UI Designs section (Figure 1)

    2. NOTE If the Application Editor is already open, bring it to the front by clicking on its tab.

    Figure 1 -The All UI Designs sectionof the UI Designs page

    The All UI section of the UI Designs Page contains a list of all UI Designs currently included in the project. The order in which the UI Designs are displayed in this list matches the order theyw ill be displayed in the tab group.

    The checked item indicates which design will be displayed when the application is started. It does not have to be the first UI Design in the list.

  2. Click the UI Design you wish to move in the tab order.

    3. The UI design is highlighted to indicate that it has been selected.

  3. Use the Move Up and Move Down buttons to change the placement of the selected UI Design in the list (and thus its tab order).
  4. Repeat to reorder other UI designs in the tab group.

    5. NOTE These changes will not become permanent until your next save of the application.uidesign file.
