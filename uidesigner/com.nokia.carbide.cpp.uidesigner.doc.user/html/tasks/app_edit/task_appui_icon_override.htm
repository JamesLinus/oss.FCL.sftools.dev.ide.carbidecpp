Overriding the AppUi Context Icon

The AppUi page allows you to set global aspects of your application interface. You may specify an icon to over-ride the AppUi context icon for a UI Design, specify a context icon explicitly for that UI Design.

Overriding the AppUi Context Icon.

  1. Open a UI Design in the UI Design Editor or select an open UI Design.

    2. The UI Design and its objects will display in the layout and non-layout regions of the UI Design Editor.

  2. Open the Status and Control drawer of the palette .

    3. The associated component types will be displayed.

    NOTE If Context Icon is not displayed as an available type the project already has a context icon specified.

  3. Drag the Context Icon into the status pane area of the layout region and release.

    4. A Context Icon is added to the UI Design's status pane.

    You may right click on the context icon to edit its image properties.
