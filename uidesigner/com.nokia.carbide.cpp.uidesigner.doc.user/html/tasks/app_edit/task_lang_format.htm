Selecting the Localization Format

Use the Languages page of the Application Editor to select the localization format for your application.

Selecting the Localization Format

  1. Select the Languages page of the Application Editor.

    2. The Languages page will be displayed, as shown in Figure 1.

    Figure 1 - The Application Editor Languages Page

  2. Click the radio button associated with the desired Localization format.

    3. NOTE The UI Designer initializes new projects to use the .loc/.lxx format. You do not need to make any changes if this is your desired format.

  3. Close the application.uidesign file.

    4. The Save Application Design confirmation dialog will appear.

  4. Click the Yes button to confirm saving the change of localization format.

    5. The UI Designer will create (or update) localization files of the appropriate type for the format you have selected.

    NOTE Existing localization files for the previous format are not deleted.
