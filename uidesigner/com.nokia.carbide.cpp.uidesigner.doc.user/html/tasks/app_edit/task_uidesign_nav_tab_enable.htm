Enabling Navigation Pane Tabbing

Navigation Pane Tabbing is easy to implement with the Carbide.c++ UI Designer.

  • Enable View Switching on the Container Name and Type Page when you create your project via the New Project Wizard.
  • Enable Navigation Pane Tabbing on the UI Designs page of the Application Editor, as described below.
  • Add or remove each UI Design in your application from the tab group as desired on the UI Designs page of the Application Editor.

Enabling Navigation Pane Tabbing

  1. Open the Application Editors UI Design page (Figure 1)

    NOTE If the Application Editor is already open, bring it to the front by clicking on its tab.

    Figure 1 -The Application Editor UI Designs page.

  2. Check the Enable Navigation Pane Tabbing check box.

    The UI Design Details controls appear when the Enable Navigation Pane Tabbing is enabled.

  3. Specify inclusion (or not) within the tab group of each UI Design in the application and set the tab or image text for each included design.

    These changes will not become permanent until your next save of the application.uidesign file.

NOTE If you uncheck the Enable Navigation Pane Tabbing check box after setting UI Design Details for any of the UI Designs, the information you set will be remembered by the UI Designer even though it will not be used in the build process until you recheck the check box.
