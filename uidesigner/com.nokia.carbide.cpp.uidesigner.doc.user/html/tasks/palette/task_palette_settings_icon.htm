Setting the Palette Icon Size

The UI Designer provides two methods to change the palette icon size. It is a matter of personal preference which one of them you use. This page describes how to set the icon size on the Palette Settings dialog. You may also set the icon size via the Palette context menu.

Figure 1 - The Palette Settings dialog showing List layout options

Setting the Palette Icon Size

  1. Open the Palette Settings dialog

    2. The Palette Settings dialog appears.

  2. Set the Use large icons options as desired (Figure 1).

    3. Checkmark the option to show large icons in the palette pane, or uncheck to show small icons.

  3. Click OK

    4. The Palette Settings dialog closes and the palette options set.
