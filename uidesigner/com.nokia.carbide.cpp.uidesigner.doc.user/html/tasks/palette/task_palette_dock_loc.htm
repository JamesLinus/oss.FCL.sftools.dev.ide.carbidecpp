Setting the Palette Dock Location

The UI Designer palette gives you access to the standard class objects which may be used to build the UI for your S60 application. The palette and its associated Component Information pane (displayed beneath the palette) extend from the top to the bottom of the UI Design View.

The palette and the layout/non-layout regions compete for the horizontal space of the UI Designer. You may place the palette to the left or right of the layout/Non-layout regions. The UI Designer provides two methods to change the palette placement.

Setting the Palette Placement via Drag and Drop.

  • Click the Palette title bar and drag the Palette to the desired location

    The cursor changes to a cross with four arrows (Figure 1) when the palette is clicked. Drag the palette to the left or right side of the UI Design Editor view to dock the palette to that side.

    Figure 1 - Preparing to drag and drop the Palette

    OR

  1. Right-click the Palette title bar to open the Palette context menu
  2. Select the Dock On > Left or Dock On > Right menu item

    3. The palette is repositioned to the chosen side of the UI Designer view.

    Figure 2 - Selecting the docking location via context menu
