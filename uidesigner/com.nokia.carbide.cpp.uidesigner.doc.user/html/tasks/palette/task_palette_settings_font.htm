Selecting the Palette Display Font

You can change the font used by the Palette pane in a UI Design Editor view. The Palette Settings dialog allows you to specify a different font or revert to the default font.

Figure 1 - The Palette Settings dialog showing Font options

Setting the Palette Display Font

  1. Open the Palette Settings dialog

    2. The Palette Settings dialog appears (Figure 1).

  2. Select the font to use in the palette pane
    • To restore the original font, click Restore Default

      • The default font is shown in the Font section of the Palette Settings dialog.

      OR

    1. To set a new font Click Change...

      2. The standard Windows Font dialog appears for selecting a font, style, and size of the Palette font (Figure 2).

      Figure 2 - The standard Windows Font dialog

    2. Click OK to close the Font dialog

      3. The chosen font is shown in the Font section of the Palette Settings dialog.

  3. Click OK

    4. The Palette Settings dialog closes and the font is applied to the palette.
