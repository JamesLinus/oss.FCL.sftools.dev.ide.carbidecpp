Showing and Hiding the Palette Drawers

Palette components are grouped into drawers of related component types. Each component drawer may be shown or hidden individually. The Palette Settings dialog allows you to specify how drawers should behave, either by always closing, closing automatically, or never closing.

 

Figure 1 - The Palette with the Editors drawer closed and opened

NOTE Some drawers contain more components than can be displayed simultaneously.

NOTE Scroll controls will be displayed at the top and/or bottom of the list of components when there is not sufficient space to display all of the components within an open drawer, as shown for the Editors drawer in Figure 1.

Showing or Hiding Palette Drawers

  1. Open the Palette Settings dialog

    2. The Palette Settings dialog appears.

  2. Select a palette Drawer option as desired (Figure 2).

    Figure 2 - The Palette Settings dialog showing the Drawer options

  3. Click OK

    4. The Palette Settings dialog closes and the palette options set.
