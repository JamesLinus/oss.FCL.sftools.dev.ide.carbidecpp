Setting the Palette Layout Options

The component Palette supports four layout options for the display of its controls (Figure 1). The display types include:

  • Columns — components are shown in columns
  • List — components are shown in a list view
  • Icons only — components are shown using only their icons
  • Details — components are shown with additional information

   

Figure 1 - The Palette layout options (column, list, icons only, details)

Setting the Palette Layout Options

  • Using the Palette context menu
    1. Right-click in the content area to show the Palette's context menu
    2. Select the desired Layout > layout_options menu item (Figure 1)

      3. The palette updates to show the selected layout option.

      Figure 2 - The Palette context menu Layout submenu

  • Using the Palette Settings dialog
    1. Right-click in the Palette view and choose Settings... from the context menu

      2. The Palette Settings dialog appears.

    2. Select a Layout option

      3. Figure 3 - Palette Settings Layout options

    3. Click OK

      4. The palette updates to show the selected layout option.
