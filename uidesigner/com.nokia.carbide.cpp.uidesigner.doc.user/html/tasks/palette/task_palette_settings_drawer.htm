Setting Palette Drawer Auto-Close Options

Each drawer of components in the component Palette may be shown or hidden individually. The Palette Settings dialog allows you to specify whether previously open drawers should auto-close when you open a new drawer.

Figure 1 - The Palette Settings dialog showing Drawer options

Setting Palette Drawer Auto-Close Options

  1. Open the Palette Settings dialog

    2. The Palette Settings dialog appears.

  2. Select the desired Drawer options setting (Figure 1).

    3. NOTE The palette will not adjust the open/closed status of any drawers until you open a new drawer after dismissing the dialog.

  3. Click OK

    4. The Palette Settings dialog closes the palette options set.
