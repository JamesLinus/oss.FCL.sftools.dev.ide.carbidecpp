Setting the Palette Icon Size

The component Palette supports the display of two icon sizes. You may use either size in any of the four layout options.

Setting the Palette Icon Size

  1. Right-click in the content area of the palette to view the Palette's context menu

    2. The Palette context menu appears (Figure 1).

    Figure 1 - The Palette context menu (showing small icons)

  2. Select the Use Large Icons menu item to toggle between showing the standard and large icons

    3. The palette updates to show the selected icon size (Figure 2).

    Figure 2- The Palette context menu (showing large icons)
