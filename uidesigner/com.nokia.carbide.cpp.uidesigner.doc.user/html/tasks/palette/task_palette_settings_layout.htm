Setting the Palette Layout Options

The palette displays two types of controls. The first set is used for selecting existing objects in your UI Designs. The second set is used to add new objects to your UI Designs.

The UI Designer supports four layout options for the display of its controls and two icon sizes, as shown in Figure 1.

   

Figure 1 - The Palette layout options (column, list, icons only, details)

The UI Designer provides two methods to change the palette layout. This page describes how to set the palette layout on the Palette Settings dialog. You may also set the layout via the Palette context menu.

Setting the Palette Layout Options

  1. Open a UI Design in the UI Design Editor or select an open UI Design.

    2. The UI Designer will be displayed with the palette either shown or hidden as per any previous settings you have made.

  2. Optional step: If the palette is hidden click the Show Palette button to show it.

    3. NOTE You cannot change the palette layout options when the palette is hidden.

  3. Right click in the content area of the Palette.

    4. The palette context menu will appear, as shown in Figure 2.

    Figure 2 - The Palette context menu

  4. Click the Settings... menu item.

    5. The Palette Settings dialog appears (Figure 3).

    Figure 3 - The Palette Settings dialog

    NOTE When the Palette Settings dialog is initially displayed it will usually cover all or part of the palette. Adjust the location of the dialog so that you can see the effects on the display of the palette contents as you adjust the settings in the dialog.

  5. Select the radio button that corresponds to the desired Layout: control option.

    6. The palette will adjust its display immediately when you select a new radio button.

  6. Click the OK button to lock in the change.

    7. The dialog will be dismissed.

    NOTE If you click the Cancel button the display will revert to the state it was in before you brought up the dialog.
