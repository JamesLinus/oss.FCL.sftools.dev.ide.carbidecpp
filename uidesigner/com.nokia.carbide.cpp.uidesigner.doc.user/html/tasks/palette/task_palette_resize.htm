Resizing the Palette

The UI Designer’s Palette can be resized vertically or horizontally. The UI Designer provides two methods to resize the palette.

Resizing the Component Palette

  • Click the divider bar between the palette and UI Designer view to select, then drag it to the desired size (Figure 1)

    • A click on the edge of the Palette converts the pointer into a horizontal bi-directional arrow indicating you can drag the divider bar to resize the Palette.

    Figure 1 - Click and drag the edge of the palette to resize

OR

  • Right-click the Palette title bar and select the Resize menu item from the context menu (Figure 2)

    • The inner side of the Palette is highlighted and the pointer changes to a horizontal bi-directional arrow and jumps to the top of that edge. Move the pointer to chosen size and release. The Palette area expands to the new size.

    Figure 2 - Right-click the Palette title bar
