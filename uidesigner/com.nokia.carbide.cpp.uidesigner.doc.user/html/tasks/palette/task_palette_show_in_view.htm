Showing the Palette as a View

The component Palette is normally displayed as a sub-pane of the UI Design Editor view, but it can be converted into a separate view (Figure 1).

Figure 1 - The Palette displayed as a separate view

Displaying the Palette in a Separate View

  1. Select the Window > Show View > Other... menu command.

    2. The Show View dialog appears (Figure 2).

    Figure 2 - The Show View dialog with the Basic group displayed.

  2. Click the tree control to display the contents of the Basic group

    3. The contents of the Basic group will be displayed as shown in Figure 2.

  3. Select the Palette item in the Basic group

    4. The Palette item will be highlighted to show it has been selected .

  4. Click OK

    5. The palette appears in a separate view (Figure 1).

    NOTE The open/closed drawer state of the palette when displayed in a view will match the previous open/closed drawer state of the palette.
