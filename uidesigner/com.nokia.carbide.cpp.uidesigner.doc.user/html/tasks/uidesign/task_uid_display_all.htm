Displaying All UI Designs

Use the Gallery view to see all the UI designs contained in your projects within the workspace. Use the Gallery view context menu to select magnification for the display and to open a selected UI design.

Figure 1 - The Gallery view

Displaying all UI Designs in the Gallery view

  1. Click Window > Show View > Gallery

    2. The Gallery view is displayed (Figure 1).

  2. Use the tree controls (gray triangle) to display or hide the UI designs for each project.
