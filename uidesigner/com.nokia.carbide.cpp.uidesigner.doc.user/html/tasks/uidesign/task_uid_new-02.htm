Selecting Initial Content

Use the S60 UI Design Selection page of the New S60 UI Design wizard to select the initial content for your UI design.

Figure 1 - Selecting the Initial Content Type

Selecting Initial Content

  1. Select the appropriate initial content type from the Select a design list
  2. Click the Next > button

    3. The Content Variants page appears if your initial content type is ListBox, else the Container Name and Type page will appear.
