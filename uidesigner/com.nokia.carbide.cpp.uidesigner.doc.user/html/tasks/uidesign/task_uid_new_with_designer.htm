Adding a UI Design to a Project

This page provides a list of the steps needed to create a new S60 UI Design using the wizard for a project that is already UI Designer enabled. The links in the discussion of each step take you to a page which contains a detailed description of the subtasks needed to perform that step.

Some common S60 UI Design Wizard actions include:

  • Click the Cancel button to terminate the wizard without creating a design
  • Click the < Back button to return to the previous page of the wizard
  • Click the Next > button to accept all entries on the current page and continue to the next page of the wizard.
  • Click the Finish button (when it is enabled) to complete the wizard.

Creating New S60 UI Designs

  1. Start the S60 UI Design Wizard

    2. Select S60 UI Design from the File > New submenu.

    The Application Properties page of the wizard will be displayed.

  2. Select the project to receive the new design

    3. Select an existing project in the current workspace from the context menu.

    The S60 UI Design Selection page of the wizard will be displayed.

  3. Select the UI Design initial content

    4. Select the initial content type for your UI design in the S60 UI Design Selection page.

    If the selected type has variants, the Content Variants page will be displayed. If it does not have variants, the Container Name and Type page of the wizard will be displayed (in which case you must skip the next step in these instructions)

  4. Select the UI Design content variant (optional step - only needed if initial content is ListBox)

    5. Select the desired variant of the initial content type in the Content Variants page.

    The Container Name and Type page of the wizard will be displayed.

  5. Set the base name and container type

    6. Set the base name and container type for the initial content in the Container Name and Type page.

  6. Resolve any name collisions (optional step - only required if a naming collision is reported)

    7. The UI Designer uses the project and UI design names to create names for the files it generates. If it detects an existing file with one of these names, a dialog will warn you of a possible name collision error and allow you to cancel or continue.

Once all of these steps are complete, a new S60 UI Design is created on disk and displayed in the Project Explorer view.
