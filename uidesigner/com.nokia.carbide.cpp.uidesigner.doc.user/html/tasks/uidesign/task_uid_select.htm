Selecting UI Designs

Before you can work with an existing UI design it must be opened and selected for editing in the UI Design Editor view.

Selecting a UI Design

  • In the Gallery view, double-click a design to open it for editing

    • The design appears in a new UI Design Editor view.

  • In the C/C++ Project view, double-click a design file (.uidesign) to open it for editing

    • The design appears in a new UI Design Editor view.

    NOTE You can select multiple .uidesign files to open by Shift- or Ctrl-clicking file names. Shift-click to select a range of .uidesign files, Ctrl-click to select discontinuous files in the project.

  • In the UI Design Editor view, click a named tab (Figure 1)

    • The UI Design Editor displays each open UI design in a named tab. Click a tab to select that UI design.
