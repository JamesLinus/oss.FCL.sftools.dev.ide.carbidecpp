Setting Magnification in the Gallery View

The Gallery View displays the UI designs for all projects in your workspace.

Set the magnification for the display of the UI designs in the Gallery View via the Gallery View context menu. The lowest magnification value (50%) allows you to view the maximum number of UI design simultaneously. The highest value (100%) allows maximum legibility of the UI design contents.

Setting UI Design Magnification in the Gallery View

  1. Click Window > Show View > Gallery

    2. The Gallery view will be displayed.

  2. Right-click in the white space area within the Gallery view

    3. The Gallery view context menu will be displayed as shown in Figure 1.

    Figure 1 - Setting Magnification in the Gallery View

    NOTE If you have selected one of the UI designs, the context menu will contain additional items as shown in Figure 2.

    Figure 2 - Setting Magnification in the Gallery View with a UI Design Selected

  3. Select the desired magnification in the context menu

    4. The UI designs will be redisplayed with the new magnification.
