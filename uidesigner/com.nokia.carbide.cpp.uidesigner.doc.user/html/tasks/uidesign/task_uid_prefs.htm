Setting UI Designer Preferences

The UI Designer preferences enable you to specify preferences for default UI Designer behaviors.

Setting UI Designer Preferences

  1. Select the Window > Preferences menu option.

    2. The Preferences window appears.

  2. Click the Carbide.c++ > UI Designer option

    3. The UI Designer preferences panel will be displayed, as shown in Figure 1.

    Figure 1 - The UI Designer Preferences panel

  3. Set the UI Designer options as desired
  4. Click OK

    5. The Preferences window closes and the UI Designer preferences saved.
