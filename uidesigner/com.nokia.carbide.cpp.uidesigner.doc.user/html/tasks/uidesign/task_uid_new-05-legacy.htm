Assigning Directories for UI Designer Files

Use the Project Directories page of the New Project Wizard to accept the default names, or provide custom names, for the directories of the generated files.

Figure 1 - Assigning project directory names

NOTE Your project probably already contains directories for each of these categories. Be careful to match the names.

NOTE If you have previously entered directory names for the associated directories using any Carbide.c++ wizard while working with any project, they will be remembered and redisplayed in this step based on the assumption that development teams will use a standard naming convention for the directories in all of their projects.

NOTE The use of standard directory names in your projects will help you avoid subtle build errors.

Specifying Project Directory Names

  1. You may accept the default directory names, or enter custom names for any or all of the directories displayed in Figure 1.
    • Include
    • Source
    • Group
    • Data
  2. When you are satisfied with the names for all directories, click the Finish button to complete the wizard.

    3. The wizard will create a new project and display it in the Project Explorer view.
