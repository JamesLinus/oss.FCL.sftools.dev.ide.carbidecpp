Starting the S60 UI Design Wizard

The S60 UI Design Wizard guides you through the steps needed to add a new UI design for your existing project.

Starting the S60 UI Design Wizard

  • Select the File > New > S60 UI Design menu item

    The New S60 UI Design window appears.
