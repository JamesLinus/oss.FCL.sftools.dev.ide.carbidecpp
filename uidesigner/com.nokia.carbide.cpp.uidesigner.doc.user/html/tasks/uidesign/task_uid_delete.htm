Deleting UI Designs

You can permanently or temporarily delete UI designs from your project.

NOTE You cannot Undo the deletion of a UI design.

NOTE If you wish to experiment with variations of your project without permanently deleting the associated UI design files, use the UI Designs Page of the Application Editor to temporarily remove the UI Design from your project. Doing so will leave the UI design in your project, but prevent it from being included in builds. You may re-import the UI design at a later time to include it in subsequent builds.

Deleting One or More UI Designs

  1. Select one or more UI Designs in the Project Explorer view.
  2. Choose one of the following methods to delete the selected UI Design(s):
    • Right-click on the selected object(s) and select the Delete option from the context menu, or

      • NOTE Be careful where you right-click to bring up the menu. A right-click directly on another item in the list will change the selection to the clicked item and display a context menu with commands that are appropriate to that item type.

    • Select the Delete option from the Edit menu, or
    • Press the Delete key

    The selected UI design(s) are removed from the project.
