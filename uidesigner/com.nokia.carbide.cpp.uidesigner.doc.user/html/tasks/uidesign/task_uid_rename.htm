Renaming a UI Design

You rename a UI design as you would any other file in your project.

Renaming a UI Design

  1. Select the UI Design in the Project Explorer view

    2. The .uidesign file will be highlighted.

  2. Right click on the selected UI Design file

    3. The Project Explorer view context menu will appear.

  3. Select the Rename option from the context menu

    4. The .uidesign file name will be editable.

  4. Modify the file name as desired and click elsewhere to accept the file name change

    5. The name change will take effect.

    NOTE You may be warned about a need to update references to the file in the application. If a dialog appears with this warning, click the Yes button to allow the UI Designer to make the necessary updates.
