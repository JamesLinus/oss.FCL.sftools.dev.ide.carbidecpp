Setting Screen Layout Configurations

S60 devices support multiple screen layout configurations. The Carbide.c++ UI Designer allows you to see how your screen layouts will appear on each of the standard supported layout configurations. Use the layout Configuration menu in the UI Design Editor toolbar to select your preferred magnification.

NOTE Screen resolution can only be set in projects with Scaleable UI, such as S60 2nd edition FP3 and above.

Figure 1 - The Layout Configuration menu in the UI Design Editor toolbar

Changing the Layout Configuration

  1. Open a UI Design in the UI Design Editor, or select a UI Design which is already open.
  2. Use the Downward Arrow button in the Layout Configuration menu to display the layout options as shown in Figure 1.
  3. Select the desired layout configuration.

    4. You may select a new layout configuration, for example:

    • S60 Landscape - 208 * 176
    • S60 QVGA Portrait - 240 * 320
    • S60 QVGA Landscape - 320 * 240
    • S60 Double Resolution Portrait - 352 * 416
    • S60 Double Resolution Landscape - 416 * 352

    The current UI Design will be redisplayed for the selected layout configuration.

    NOTE The change will affect only the currently selected UI Design.
