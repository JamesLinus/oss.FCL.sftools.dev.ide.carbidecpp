Setting UI Design Magnifications

You can adjust the magnification of the screens that are displayed within the layout region of the UI Design Editor. Use the UI Design Magnification menu in the UI Design Editor toolbar to select your preferred magnification.

Figure 1 - The UI Design Magnification menu in the UI Design Editor toolbar

NOTE Most of the UI Design Editor toolbar, including the magnification menu, is enabled only when a UI design has focus in the UI Design Editor.

Changing the UI Design Display Magnification

  1. Open a UI Design in the UI Design Editor, or select a UI Design which is already open.
  2. Use the Downward Arrow button in the Magnification menu to display the magnification options as shown in Figure 1.
  3. Select the desired magnification level.

    4. You may select from the following magnification levels:

    • 50%
    • 75%
    • 100%
    • 200%
    • 400%

    The current UI design will be redisplayed at the selected magnification level.

    NOTE The change will affect only the currently selected UI design.
