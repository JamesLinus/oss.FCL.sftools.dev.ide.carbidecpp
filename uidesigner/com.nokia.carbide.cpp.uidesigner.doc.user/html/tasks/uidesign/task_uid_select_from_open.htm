Selecting an Open UI Design in the UI Design Editor

Before you can work with an existing UI design it must be selected and displayed at the front of the UI Design Editor.

The UI Design Editor displays each open UI design in a named tab. Click a tab to select that UI Design and bring it to the front.

Figure 1 - Tabbed display of open UI Designs

Selecting an open UI Design

  1. Scroll the list of tabs (if necesssary) to display the name of the desired UI design.

    2. NOTE Scrolling will be implemented automatically when more UI designs are open than names can be displayed in the available tab space. Figure 1 shows the number 4 in the far right section, indicating that 4 more UI designs may be scrolled to in that direction.

  2. Click on the tab for the UI Design you wish to select.

    3. The UI design will be brought to the front of the UI Design Editor.
