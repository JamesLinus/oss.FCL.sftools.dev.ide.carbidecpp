Selecting the Project

The S60 UI Design Wizard guides you through the steps needed to add a new UI Design for your existing project. Use the Application Properties page (Figure 1) to select the project for your new UI Design. If the project does not already support UI designs, additional information is requested (Figure 2). Upon successful completion of the wizard, the project is enabled for UI designs.

Selecting the Project for the New Design

  • If the project is already UI Designer enabled (Figure 1):
    1. Select an existing project from the Add to project context menu

      2. Figure 1 - Selecting the project

    2. Click the Next > button

    • The S60 UI Design Selection page appears.

OR

  • If the project is not UI Designer enabled (Figure 2):
    1. Select an existing project from the Add to project context menu
    2. Select which SDK to use from the Design for baseline SDK context menu
    3. Specify the Initial language setting

      4. Figure 2 - Application Properies for non-UI Designer projects

    4. Click the Next > button

      5. The S60 UI Design Selection page appears.
