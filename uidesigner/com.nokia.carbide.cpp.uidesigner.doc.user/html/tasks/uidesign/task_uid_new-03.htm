Selecting The Content Variant (optional)

The List Box container type requires additional information before it can be fully defined for initialization by the UI Designer. Use the Content Variants Page of the New S60 UI Design wizard to select the desired variant of your initial content. It will be displayed automatically if you select an initial container type of List Box.

Figure 1 - Selecting the Content Variant

NOTE This screen will only appear if your initial content type is List Box.

Selecting the Content Variant

  1. Select the list box type from the Select the type of list box list
  2. Click the Next > button

    3. The Container Name and Type page appears.
