Deleting a Menu

Use UI Designer to delete any menu and all its submenus and menu items from the project.

    Figure 1 - A menu component selected in the UI Design Editor page and Outline view

Deleting an Existing Menu

  1. Open the UI design that contains the menu component to delete
  2. Select the menu component in the non-layout region or in the Outline view
  3. Delete the menu component by:
    • Right-click the menu component in the non-layout region, then select Delete from the context menu

      • OR

    • Press the Delete key

    The selected menu component is deleted from the project.
