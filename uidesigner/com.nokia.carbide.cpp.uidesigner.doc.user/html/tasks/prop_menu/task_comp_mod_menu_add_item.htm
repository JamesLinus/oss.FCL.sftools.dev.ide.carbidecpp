Adding a Menu Item

UI Designer provides an intuitive interface for adding menus and menu items to your UI designs. When a menu component is placed into a UI design it is represented by an icon in the non-layout region of the UI Design Editor view. The editing of menu items is done using the menu editor.

Adding a New Item to an Existing Menu

  1. Open a UI Design in the UI Design Editor or select an open UI Design.

    2. If this UI Design contains any Menu Bar components, they will be displayed in the non-layout region. If the design does not contain any existing menus, you must first add the menu to this design before you can add items to it.

    Figure 1 shows a UI Design which contains one Menu Bar, named menuBar1.

    Figure 1 - A UI Design which contains a single Menu Bar, menuBar1.

  2. Click the icon for the menu bar in the non-layout region to select it.

    3. As shown in Figure 2, the menu will be displayed at the bottom of the content region of your UI Design. Any existing items in this menu will be displayed. A place for a new item will be indicated by the text "Type here".

    Figure 2 - A selected menu is ready for the addition of new items.

  3. Optional - Click on the "Type here" text to enable editing of the placeholder for the new menu item.

    4. An empty edit box appears over the item.

    NOTE You do not need to perform this step to enable editing of the menu item. All text typed while the "Type here" item is displayed will replace the "Type here" item name.

  4. Type the name of the menu item into the edit box.
  5. Optional - Press the ENTER key to accept the name of the new item and initialize a new potential item named "Type here".

    6. You can repeat steps 3 through 5 as many times as necessary to add additional items to the menu.

  6. Click anywhere EXCEPT within the menu to accept all edits performed in this session.

    7. The menu will no longer be displayed, but your additions/changes have been saved.
