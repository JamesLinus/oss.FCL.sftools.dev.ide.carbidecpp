Adding a Submenu

UI Designer provides an intuitive interface for adding menus and menu items to your UI designs. Any existing top level menu item can become the parent of a submenu simply by typing an entry into the submenu text edit field.

Adding a Submenu to an Existing Menu Item

  1. Open a UI Design in the UI Design Editor or select an open UI Design.

    2. If this UI Design contains any Menu Bar components, they will be displayed in the non-layout region.

    NOTE If the design does not contain any existing menus, you must first add the menu to this design and then add items to the menu before you can add a submenu to it.

    Figure 1 shows a UI Design which contains one Menu Bar, named menuBar1.

    Figure 1 - A UI Design which contains a single Menu Bar, menuBar1.

  2. Click the icon for the menu bar in the non-layout region to select it.

    3. As shown in Figure 2, the menu will be displayed at the bottom of the content region of your UI Design. Any existing items in this menu will be displayed. A place for a new item will be indicated by the text "Type here".

    Figure 2 - A selected menu is ready for the addition of new items.

  3. Click the existing menu item to which you wish to add a submenu.

    4. A submenu will appear to the right with the first item labeled "Type here", as shown in Figure 3.

    Figure 3 - A selected menu is ready for the addition of new items.

  4. Click in the submenu to enable editing of its contents.

    5. The 'Type here' text will be displayed in an edit-enabled box.

  5. Continue adding items to the submenu just as you add items to the menu.
