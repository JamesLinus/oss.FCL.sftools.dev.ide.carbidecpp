Renaming a Menu Item

UI Designer provides an intuitive interface for adding menu support to your UI Designs.

Any menu item can be renamed at will.

Renaming an Existing Menu Item

  1. Open a UI Design in the UI Design Editor or select an open UI Design.

    2. If this UI Design contains any Menu Bar components, they will be displayed in the non-layout region.

    Figure 1 shows a UI Design which contains one Menu Bar, named menuBar1.

    Figure 1 - A UI Design which contains a single Menu Bar, menuBar1.

  2. Click the icon for the menu bar in the non-layout region to select it.

    3. As shown in Figure 2, the menu will be displayed at the bottom of the content region of your UI Design. Any existing items in this menu will be displayed. A place for a new item will be indicated by the text "Type here".

    Figure 2 - A menu which contains three items.

  3. Click the existing menu item which you wish to rename.

    4. The item will be displayed in a rectangle, as shown in Figure 3.

    NOTE If the item has a submenu it will be displayed.

    Figure 3 - A selected menu item is ready to be renamed.

  4. Type in the text you wish for the menu item.

    5. The item label will update to the entered text.

  5. Click elsewhere in the layout region to accept the item label.

    6. The item label will be changed to the new value.
