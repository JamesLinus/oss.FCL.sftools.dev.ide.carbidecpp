Reordering Menu Items

UI Designer provides an intuitive interface for adding menu support to your UI Designs.

Any menu item can be reordered at will.

Reordering Menu Items

  1. Select the menu item to move in the Outline view.

    2. The menu item is highlighted.

  2. Drag the menu item within the menu in the Outline view.

    3. A dark horizontal bar appears when the menu item is in a location to which it can be moved.

  3. Release the mouse button when the menu item is in the desired location.

    4. The menu item is displayed in its new location.
