Adding Menus

UI Designer provides an intuitive interface for adding menu support to your UI designs. When a menu component is placed into a UI design it is represented by an icon in the non-layout region of the UI Design Editor view. The editing of menu items is done using the menu editor.

Adding a Menu to a UI Design

  1. Open a UI Design in the UI Design Editor or select an open UI Design.
  2. Open the Menu Drawer of the Palette.
  3. Drag a Menu Bar component into the layout or non-layout region of your UI design, or into the Outline view.

    4. The cursor displays a plus sign () inside of a small square when it is located at a place which will accept the new Menu Bar component or a slashed circle () when it is located at a place that will not accept it.

  4. Release the mouse button while the cursor is displaying the plus sign inside of a square.

    5. A new menu is added to the design. As shown in figure 1, the new menu initializes the text for the first item to the string "Type here".

    Figure 1 - A newly created menu inviting you to type the name of its first item.

    You can now rename its first entry and/or add additional items to the menu.
