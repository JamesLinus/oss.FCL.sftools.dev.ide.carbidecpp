Editing Text Data

Text data is entered by typing the desired value directly into the component of using the associated field in the Properties view. For components that support scalable strings the Edit String Data dialog is provided.

Editing Text Data

  1. Select an object which contains text data.

    2. The object will display a bounding rectangle with squares at its corners and midpoints.

  2. Display the Properties view.

    3. The properties of the selected object will be displayed.

  3. Click on the value of any text data which you wish to modify

    4. The value will be highlighted and display a flashing cursor.

  4. Edit the value of the field using standard edit operations.

    5. You may cut, copy and paste any values which are legal for the field.

    NOTE Monitor the Carbide.c++ status pane as you enter data for any reported errors.

  5. Exit the text field to save your work

    6. Press the Tab or any Arrow key, click outside the text field, select a different field for editing, or click in any other view. If the value you entered is legal for this field, it will be accepted.
