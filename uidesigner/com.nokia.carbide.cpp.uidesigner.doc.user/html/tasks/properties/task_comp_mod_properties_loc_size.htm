Editing Location/Size Data

Size/location data is defined as a pair of x & y co-ordinates. When specifying location information these values determine the upper left corner of the object's location. When specifying size information these values determine the width and height of the object. The Properties view provides a pair of numeric editors to accept the individual x & y values.

Values for the x & y co-ordinates must be in the range from -1000 to 1000.

NOTE It is possible to enter legal values within that range which will result in your object being displayed 'offscreen'.

Editing Size/Location Data

  1. Select an object which contains size/location data.

    The object will display a bounding rectangle with squares at its corners.

  2. Display the Properties view.

    The properties of the selected object will be displayed.

  3. Optional: Click on the [+] tree control to the left of the name of any location or size data which you wish to modify (if the field is not already disclosed).

    The x & y subfields will be displayed, as shown in Figure 1.

    Figure 1 - A layout field with location and size subfields

    NOTE Some fields will display descriptive text in the Carbide.c++ status pane when they are selected.

  4. Edit the value of the x and/or y fields using standard edit field operations.

    You may cut, copy and paste any values which are legal for the field.

    NOTE Monitor the Carbide.c++ status pane as you enter data for any reported errors.

  5. Press the Tab key, click the mouse in another area of the Properties view to select a different field for editing, or click in any other view.

    If the value you entered is legal for this field, it will be accepted.
