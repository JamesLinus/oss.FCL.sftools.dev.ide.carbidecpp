Editing Image Property Data

To use an image in an application it must be present in the project. Images are normally stored in Multi-Image Files (MIF) and MultiBitMap (MBM) files contained within the project. If an image is not in the project, it must be added before you can use it.

The two methods of accessing image component properties include:

  • If the project's Multi-Image File contains the image, use the Edit Image Property dialog or Edit Multi-Image Property dialog to select the image.
  • If the desired image is not in the current Multi-Image File, click the Edit MBM/MIF File... button in the Edit Image Property dialog to add the image to the project by clicking the Add From Project or Add From Filesystem button in the Edit MBM/MIF Contents dialog.

Editing Image Property Data

  • Right-click on the image component and select Edit Image Property

    • OR

  • Click the image's (browse) control in the Properties view

    • The Edit Image Property dialog or Edit Multi-Image Property dialog appears.
