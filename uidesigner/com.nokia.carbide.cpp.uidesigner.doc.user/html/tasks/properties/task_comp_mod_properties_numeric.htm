Editing Numeric Data

Numeric data is entered by typing the desired value directly into the associated field. No special editor is provided for this type of data.

Components which manage numeric data allow you to specify a minium-maximum range for the value of the data which the user may enter.

Editing Numeric Data

  1. Select an object which contains numeric data.

    2. The object will display a bounding rectangle with squares at its corners and midpoints.

  2. Display the Properties view.

    3. The properties of the selected object will be displayed.

  3. Click on the value of any numeric data which you wish to modify

    4. The value will be highlighted and display a flashing cursor.

    NOTE Some fields will display descriptive text in the Carbide.c++ status pane when they are selected.

  4. Edit the value of the field using standard edit field operations.

    5. You may cut, copy and paste any values which are legal for the field.

    NOTE Monitor the Carbide.c++ status pane as you enter data for any reported errors.

  5. Press the Tab key, click the mouse in another area of the Properties view to select a different field for editing, or click in any other view.

    6. If the value you entered is legal for this field, it will be accepted.
