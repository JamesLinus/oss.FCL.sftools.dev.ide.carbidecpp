Adding Images

If the image you wish to use is not in a project MBM or MIF file already contained within a project directory, use the Edit MBM/MIF Contents dialog to add it to the selected multi-image file as described below.

Adding Bitmap or Icon Images

  1. Invoke the Edit MBM/MIF Contents dialog.

    NOTE This dialog is invoked by selecting the Edit MBM/MIF File... button in the Edit Image Property or Edit Multi-Image Property dialog. This dialog is slightly different depending on whether a MBM or MIF file was selected when the Edit MBM/MIF File... button was selected. Click the Add From Project or Add From Filesystem button to add a graphics file to the selected multi-image file.

    The Edit MBM/MIF Contents dialog will appear, as shown in Figure 1.

    Figure 1 - The Edit MBM or MIF Contents dialog

    The dialog displays a list of all appropriate graphic file types which are contained within the project's directories.

  2. [Optional] Click the Add From Project or Add From Filesystem button if the desired image is not listed.

    Use the Add Images From Project or Add Images From Filesystem dialogs to add images to the previously selected multi-image file. When you return from the dialog the images you added will be listed in the Edit MBM/MIF Contents dialog.

    Figure 2 - The Add Images From Filesystem dialog

  3. In the Edit MBM/MIF Contents dialog, select the desired bitmap file, mask file, or icon file as appropriate to the dialog.

    Color and bit depth information for the source file will be displayed.

  4. Set the desired color and bit depth(s).
  5. Click the OK button.

    This dialog will be dismissed. You will be returned to step 4 of the Selecting Images task. You may select the image you have just added.
