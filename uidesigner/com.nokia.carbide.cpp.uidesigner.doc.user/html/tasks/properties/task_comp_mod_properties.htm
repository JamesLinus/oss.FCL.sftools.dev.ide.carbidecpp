Inspecting/Modifying Object Properties

The UI Designer's Properties view provides a standard method to edit the properties associated with the elements of your application. All supported data types have custom editors to help guide you in entering correct values for each property.

This section describes the specific steps needed to examine and/or modify the object properties.

Topics in this section include:

  • Editing Control Button Arrays
  • Editing Date Data

  • Editing Image Property Data
  • Editing Location/Size Data
  • Editing Numeric Data

  • Editing Text Data
  • Editing Time Data
  • Restoring Default Values
