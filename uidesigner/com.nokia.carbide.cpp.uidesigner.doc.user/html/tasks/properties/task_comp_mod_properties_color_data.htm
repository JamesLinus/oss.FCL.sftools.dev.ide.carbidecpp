Editing Color Data

You may work with 64 distinct colors within your application of which 48 colors are pre-defined.

You may set the final 16 to any value, using either HSL or RGB descriptors.

Editing Color Data

  1. The Color dialog is invoked when you need to edit any color property.

    2. The Color dialog is displayed as shown in Figure 1 .

    Figure 1 - The Color dialog.

  2. Optional: If the desired color is not in the Basic or Custom colors click the Define Custom Colors >> button.

    3. The 'Expanded' Color dialog will be displayed as shown in Figure 2.

    Figure 2 - The 'Expanded' Color dialog.

  3. Optional (perform this step if defining a custom color): Click one of the 16 Custom colors to receive the selected color.

    4. NOTE If you do not specify a location, the editor will attempt to assign the new color to the 'next' location. This selection could over-write a desired custom color.

  4. Optional (perform this step if defining a custom color): Use the standard color definition settings to specify the desired color using HSL or RGB values.

    5. The color will be displayed within the Color/Solid rectangle.

  5. Optional (perform this step if defining a custom color): Click the Add to Custom Colors button.

    6. The defined color will be displayed within the selected custom color rectangle.

  6. Click the desired color from the displayed Basic or Custom colors.

    7. The clicked color will be displayed with a bounding rectangle .

  7. Click the OK button.

    8. The Color dialog will be dismissed.

    The selected color value will be assigned to the object which invoked the Color dialog.
