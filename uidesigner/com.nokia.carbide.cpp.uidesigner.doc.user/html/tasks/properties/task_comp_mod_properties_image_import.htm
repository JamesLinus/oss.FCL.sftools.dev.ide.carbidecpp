Importing Images

If the image you wish to use is not in a project multi-image file and is not contained within a project directory, use the Import dialog to import it into a project directory as described below and then use the Add Bitmap or Add Icon dialog to add it to the selected multi-image file.

Importing Images

  1. Invoke the Import dialog.

    2. NOTE This dialog is invoked in step 2 of the Adding Images task. This dialog is displayed after you click the Import bitmaps... button. This dialog is used to import a graphics file into the project directory.

    The Import dialog will appear, as shown in Figures 1.

    Figure 1 - The Import dialog

  2. Use the top Browse... button to navigate to the directory which contains the graphics file you wish to import.

    3. The directory and its graphics files will be displayed in the From directory: and the listing panes.

  3. Check the boxes for each file you wish to import.
  4. Use the bottom Browse... button to navigate to the directory within your project to which you wish to import the graphics files.

    5. The directory name will be displayed in the Into folder: pane.

  5. Click the Finish button.

    6. The selected files will be copied into the selected project folder.

    This dialog will be dismissed. You will be returned to step 2 of the Adding Images task. You may select the image you have just added.
