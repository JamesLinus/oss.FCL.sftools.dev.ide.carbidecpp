Editing Time Data

Time data is defined by specifying complete time values in the form of HR:MN:SC xM, or by entering hour, minute, and/or second data individually.

You may provide maximum and minimum times which the user may enter, as well as specifying the starting value for the field.

You may set flags which specify whether the field should display its hour and/or second data.

Editing Time Data

  1. Select an object in a UI Design which contains time data.

    2. The object will display a bounding rectangle with squares at its corners and midpoints.

  2. Display the Properties view.

    3. The properties of the selected object will be displayed.

  3. Optional: Click on the [+] tree control to the left of the name of any time field which you wish to modify (if the field is not already disclosed).

    4. The flags, maximum time, minimum time, and time subfields will be displayed.

  4. Optional: Click on the [+] tree control to the left of the name of any flags, maximum time, minimum time, and time subfields subfields which you wish to modify (if the field is not already disclosed).

    5. The hide hours field, hide seconds field, hour, minute, and seconds subfields will be displayed, as shown in Figure 1.

    Figure 1 - A time field with all of its subfields disclosed for editing.

    NOTE Some fields will display descriptive text in the Carbide.c++ status pane when they are selected.

  5. Edit the value of any hide hours field*, hide seconds field*, maximum time, minimum time, time, hide hours field, hide seconds field, hour, minute, and seconds subfields using standard edit field operations.

    6. You may cut, copy and paste any values which are legal for the field.

    NOTE *The hide hours field and hide seconds field subfields provide an enumerated type editor. You cannot type or paste data into those subfields.

    NOTE The maximum time, minimum time, and time subfields require you to enter time data using AM/PM notation. The hour subfiled requires you to enter data using 24-hour-clock notation.

    NOTE Monitor the Carbide.c++ status pane as you enter data for any reported errors.

  6. Press the Tab key, click the mouse in another area of the Properties view to select a different field for editing, or click in any other view.

    7. If the value you entered is legal for this field, it will be accepted.
