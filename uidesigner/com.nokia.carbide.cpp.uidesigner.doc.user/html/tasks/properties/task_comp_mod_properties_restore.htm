Restoring Default Values

The Properties view provides a convenient method to restore any value you have edited to its default value.

Restoring Default Values

  1. Select an object in a UI Design for which you wish to restore one or more default values.

    2. The object will display a bounding rectangle with squares at its corners and midpoints.

  2. Display the Properties view.

    3. The properties of the selected object will be displayed.

  3. Right-Click in the Property area for the field you wish to restore.

    4. The Properties view context menu will appear.

  4. Select the Restore Default Value item from the context menu.

    5. NOTE This menu item will be disabled if the associated field already contains its default value.

    The value of the field data will be reset to its default value.
