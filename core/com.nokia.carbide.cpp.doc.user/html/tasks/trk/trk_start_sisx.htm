Starting the CODA Software

The location of the CODA (Carbide On-device Debug Agent) software varies based on the phone. On some phones it will be the Installed, Applications, or the My Own folder. Refer to the documentation that came with the phone to learn where installed programs are stored.

  1. Make sure CODA (Carbide On-device Debug Agent) is present on the device

    To install CODA see Install Remote Agents tab in the New Connections dialog.

  2. Launch the CODA program on the device
  3. Verify the connection is working on the device and within Carbide

      

    Figure 1. Connection verified on device and Carbide

  4. If CODA reports "Not Connected"
    1. Click Options > Settings soft key
    2. If Connection is not set to USB, choose USB

    3. Click Back soft key
    4. Restart CODA
