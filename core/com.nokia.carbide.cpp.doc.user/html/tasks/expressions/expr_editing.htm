Editing Expressions

You can edit any valid C/C++ expression in the Expressions view using Edit Watch Expression.

Editing Expressions

  1. Right-click an expression and select Edit Watch Expression

    2. The Edit Watch Expression window (Figure 1) appears.

    Figure 1 - Edit Watch Expression window

  2. Modify the expression in the Expression text box.
  3. Click OK.

    4. The modified expression appears in the Expressions view.
