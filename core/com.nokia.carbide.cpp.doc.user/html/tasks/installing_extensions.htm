Installing Extensions

Use the Install Extensions view to find, select, download, and install Carbide extensions to your work environment.

Installing one or more extensions

  1. In the Install Extensions view checkmark one or more extensions
  2. Click View Menu () > Install Checked Items option

Carbide then downloads and installs the selected extensions to your work environment. A restart of Carbide may be required to completely load the new features.
