Stop Mode On-Device Debugging Preconditions

Before conducting a Stop Mode On-Device debug session the following preconditions should be met.

Verifying On-device Debugging Preconditions

  • Install RVCT 4.0 or greater compiler (if required)
  • Install Simulator (if required)
  • Verify that your SDK or customer kit is recognized in the Symbian SDK preference panel
  • Build your ROM Image if required
