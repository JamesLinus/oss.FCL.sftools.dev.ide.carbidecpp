Run Mode On-Device Debugging

To implement run mode on-device debugging the following tasks need to be completed.

  1. Install device connection software
  2. Install Perl
  3. Install an SDK
  4. Setup environment variables
  5. Setup a virtual drive for Techview and Cust Kits
  6. Set the default kit in Devices.xml
  7. Install the SISX file on the target device
  8. Configure CODA connection on the device
  9. Create a launch configuration

Configure CODA Connection on the Device

Connecting with Nokia Ovi/PC Suite
  1. Verify Nokia Ovi/PC Suite are installed on the PC (drivers are required by Carbide)
  2. Connect the USB cable to the PC and then to the target device
  3. On the device select PC Suite from the USB mode list
  4. On the device launch the CODA agent

    5. The location of the CODA on-device debug agent will vary based on the phone. On some phones it will be in the Installed, Applications, or My Own folder.

  5. Click Options > Settings > Connection
  6. Change Connection type to USB and set connection options

  7. Select Back and start the CODA software again and you should be connected
  8. On the PC verify connection in Remote Connections view

With the remote device connected you are ready to start debugging.
