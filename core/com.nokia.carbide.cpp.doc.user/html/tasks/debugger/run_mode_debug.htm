Run Mode On-Device Debugging

To implement run mode on-device debugging the following tasks need to be completed.

  1. Install device connection software
  2. Install Perl
  3. Install an SDK
  4. Setup environment variables
  5. Setup a virtual drive for Techview and Cust Kits
  6. Set the default kit in Devices.xml
  7. Install the SISX file on the target device
  8. Configure TRK connection on the device
  9. Create a launch configuration

Install Device Connection Software

The two choices for connecting are Bluetooth and PC Suite.

BlueTooth
  1. System requirements:
    • Administration rights on your computer
    • BT dongle attached to your computer
    • Bluetooth installer to install the drivers
  2. Install the Bluetooth dongle to a USB port
  3. Install the drivers from the Bluetooth CD
  4. Click on the Bluetooth icon in your system tray and run the Bluetooth setup
  5. Right click on the Bluetooth icon in the System tray and select Advanced Configuration
  6. Click the Accessibility tab
    1. Check the option “Let other Bluetooth devices discover this computer”
    2. Set the option under “Allow” to “All Devices”

NOTE The Nokia PC image includes the Microsoft Bluetooth drivers which do not support serial emulation over Bluetooth. You need to download and install the drivers provided by the Bluetooth manufacturer to connect to a device.

Nokia PC Suite 6.8
  1. Install PC Suite (or later version)

    Launch the PC Suite installer and follow the onscreen instructions to install the software. When finished, a Get Connected wizard should appear

  2. Press Next
  3. In the "Select your connection type" window select "Cable Connection" and press Next
  4. You will be prompted to connect the phone via the USB cable
  5. Connect the phone and continue with onscreen instructions
  6. After installation your phone should be listed
  7. Click Finish
  8. Select PC suite on your phone

    9. The Nokia PC Suite window should appear

  9. Select the Manage Connections icon
  10. Uncheck all the connections available
  11. Click Ok

    12. PC Suite is now ready to use.
