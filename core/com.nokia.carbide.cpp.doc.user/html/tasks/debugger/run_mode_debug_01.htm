Installing Perl

Perl is a high-level, general-purpose, interpreted, dynamic programming language which many Symbian OS build utilities and commands require.

Installing Perl

  • Download and install ActivePerl version 5.6.1 b638

Software can be downloaded from Forum Nokia.

  • BEFORE 31-DEC-2010: ActivePerl-5.6.1.635 from Forum Nokia
    (5.6.1 is required by Symbian tools, later versions will not work)
  • AFTER 31-DEC-2010: All SDKs will include Perl as part of the installer.
    Consult the Forum Nokia or the Symbian Foundation sites for details.
