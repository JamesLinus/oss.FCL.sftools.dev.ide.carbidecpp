Run Mode On-Device Debugging

To implement run mode on-device debugging the following tasks need to be completed.

  1. Install device connection software
  2. Install Perl
  3. Install an SDK
  4. Setup environment variables
  5. Setup a virtual drive for Techview and Cust Kits
  6. Set the default kit in Devices.xml
  7. Install the SISX file on the target device
  8. Configure CODA connection on the device
  9. Create a launch configuration

Install the SISX File on the Target Device

The on-device debug agent is delivered to the target device as a SISX file. There are two versions of CODA: Application CODA and System CODA.

  • Application CODA— supports application debugging only and available to developers creating applications for devices. Developers do not have access to system level routines, system ROM, and other user application code. User cannot modify the LR and SR registers, and must use a SIS file to debug an application.
  • System CODA— supports both application and limited ROM debugging; including application executables and limited ROM based components (no device drivers). Also available to original equipment manufacturers (OEMs) for device creation and customization.

The following SISX file is officially signed by Nokia: s60_3_0_app_trk_2_6_nokia_signed.sisx

Installing using Bluetooth
  1. Right click on the Bluetooth item on the system tray and select Quick Connect > File Transfer > Find Devices
  2. Refresh the File Transfer window to allow the device to detect all the Bluetooth devices
  3. From the list select your device and click Open
  4. You should enable the My Bluetooth Places window popup option that creates a connection shortcut “MyDeviceOBEX File Transfer” in the My Bluetooth Places folder.
  5. The Obex File Transfer window for your devices should open from within the My Bluetooth Folder
  6. You will be prompted to enter a Bluetooth PIN code. Enter any number.
  7. On the device you will be prompted to enter the same PIN code. Enter PIN code.
  8. The device will then ask you to accept the connection request from the PC. Press OK.

    9. The OBEX folder on the PC should now be available.

  9. Drop the CODA SISX file into the folder
  10. The device will prompt you to accept the connection request from the PC. Press OK.
  11. The SISX file should then transfer from the PC to the device
  12. Navigate to where the SISX file was placed

    13. If you dropped it into the OBEX folder it will be in Applications > Messages > Inbox. If you dropped it into the C folder it will be in Applications > Utilities >FileBrowse > C.

  13. Select the CODA SISX file on the device to install the CODA debug agent.
Installing using PC Suite
  1. Place the SISX file on your PC
  2. Connect the USB cable to the phone
  3. On the device it will display USB mode:
    PC Suite
    File transfer
  4. Select PC Suite and the device will connect to the PC
  5. Open PC suite. In the bottom left a status display should indicate “Series 60 Device connected via USB”.
  6. Double click “Install applications”. The Nokia Application Installer wizard will appear.
  7. On the left you should see My Computer and on the right My Phone
  8. From My Computer select the SISX file on your PC
  9. From My Phone select the “Phone memory” drop down list an select where you want to install the file.
  10. Select the green arrow pointing from My Computer to My Device. The SISX file should then be transferred to the device and installed.
