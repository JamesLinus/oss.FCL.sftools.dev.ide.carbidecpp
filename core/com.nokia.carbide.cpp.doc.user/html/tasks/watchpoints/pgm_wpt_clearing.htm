Clearing Watchpoints

Use the Toggle Watchpoint menu item to clear a watchpoint.

Clearing a Watchpoint

  • Select the global variable to clear the watchpoint on

    • The global variable is highlighted in the source file.

    NOTE If a global variable is not selected the Toggle Watchpoint menu item will remain disabled.

  • Click the Run > Toggle Watchpoint menu item

    • The watchpoint is removed from the Breakpoints view list, and the appropriate watchpoint status icon disappears from the marker bar next to the global variable.
