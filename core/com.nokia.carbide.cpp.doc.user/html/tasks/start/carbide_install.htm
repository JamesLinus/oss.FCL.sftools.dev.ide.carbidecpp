Carbide.c++ Installation

You must install Carbide.c++ on a computer in order to create Symbian programs.

NOTE Also, see Carbide.c++ Requirements to ensure you have everything required to develop Symbian programs.

Installing Carbide.c++

  1. Launch the Carbide.c++ installer

    2. The first page of install wizard appears.

  2. Follow install wizard instructions and accept default settings
  3. Click Finish in installer window

    4. The installer exits.

  4. Install a software development kit (SDK) for Symbian development

    5. You must have at least one Symbian OS SDK's installed to develop programs for Symbian devices. See System Requirements for a complete list of supported SDKs.

  5. Install Perl

    6. Download and install ActivePerl-5.6.1.635.

  6. Install the “Symbian Connect” software that came with your phone

    7. You need software that communicates with the target device from your PC. For more information, see Deploying Programs to Devices.
