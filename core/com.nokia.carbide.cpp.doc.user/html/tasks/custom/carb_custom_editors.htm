Customizing Carbide: Editors

You can customize how the Carbide C/C++ editor looks and behaves to suit your work style.

The C/++ > Editor preferences include:

  • Editor
  • Content Assist
  • Folding
  • Hovers
  • Mark Occurences
  • Save Actions
  • Scalability
  • Syntax Coloring
  • Templates
  • Typing
