Adding an SDK

The Symbian OS plug-ins can usually detect when a SDK has been installed, as described in SDK Support.

To do this:

  1. Click Window > Preferences to open the Preferences window, then select SDK Preferences in the Carbide.c++ group to show the page for SDKs.

  2. Click Add New SDK to add a specific SDK or Rescan All SDKs to search the installed SDKs and add them to the Available Symbian OS SDKs list.

  3. Click OK to save the settings.
