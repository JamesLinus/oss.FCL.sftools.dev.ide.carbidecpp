Adding SDKs

Use the Add New SDK dialog to provide the information Carbide.c++ requires to identify and locate the specified SDK.

Figure 1. Add New SDK dialog

Table 1 Add New SDK dialog—items
Item Explanation
SDK ID The name used to identify this SDK.
EPOCROOT The path to EPOCROOT within the SDK. Click Browse to locate and select EPOCROOT.

Adding SDKs to Carbide.c++

  1. Click Add New SDK in the Symbian SDKs preference panel

    2. The Add New SDK window (Figure 1) appears.

  2. Enter the identifying SDK ID
  3. Enter the EPOCROOT path or click Browse to locate it
  4. Click OK

    5. The Add New SDK dialog closes and the SDK is added to the Symbian SDKs list.
