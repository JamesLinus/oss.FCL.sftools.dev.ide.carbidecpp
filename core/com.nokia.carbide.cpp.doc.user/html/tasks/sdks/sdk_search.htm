Searching for SDKs

Use the Symbian SDKs panel to search for installed Symbian OS SDKs.

Figure 1. Symbian SDKs panel

Searching for SDKs

  • Click Rescan All SDKs in the Symbian SDKs panel

    Carbide rescans the hard drive searching for install SDKs, adding them to the Available Symbian OS SDKs list.
