Exporting Remote Connections

You can export remote connection settings for sharing and import settings created by others using the Import/Export commands.

  • Exporting Remote Connections
  • Importing Remote Connections

To export one or more remote connections:

  1. Click the File > Export... menu option.
  2. Select the Remote Connections > Remote Connections Data wizard in the Select page (Figure 1), then click Next.

    3. Figure 1. Select page showing Remote Connections Data wizard

  3. Choose file and remote connections to export (Figure 2) in the Export Remote Connections Data page.

    4. Figure 2. Enable remote connections to export to chosen file

  4. Click Finish to save the selected remote connections to the file.
