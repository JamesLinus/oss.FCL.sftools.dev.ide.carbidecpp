CODA Pane Overview

When debugging a remote device using CODA, the following panes appear in the launch configuration window:

  • Main Pane
  • Debugger Pane
  • File Transfer Pane
  • Executables Pane
  • Installation Pane
  • ROM Log (Symbian OS Device only)
  • Source—specifies the location of source files used when debugging a C or C++ application
  • Common—specifies the location to store your run configuration, how to access it and what perspective to open when running an application
