System Search view

The System Search view displays the results of a search based on the criteria used in the System Search page. This view (Figure 1) appears automatically when a directory search is performed. You can also open it using the Window > Show View > Other... option where it is found in the Show View dialog under the System Search category.

Figure 1. System Search view after a system search

The System Search view displays these columns:

  • Name — name of the matched file and the number of matches found within it
  • Type — the file type extension (.c, .c++, etc.)
  • Size — size of the file in bytes
  • Date Modified — last date the file was modified
  • Path — complete path to locate the file

These columns are sortable, simply click the column title to sort in ascending or descending order. Most reordering is based on strings and will be instant, but sorting by Size may take longer depending upon how many file matches have been made.

System Search view toolbar icons

The table below lists the icons displayed in the System Search view toolbar.

Icon Item Explanation
Show Next Match

Opens the file in an editor if not already open and shows the first match in the file. If an editor is already open then show the next match in the file.

Show Previous Match Opens the file in an editor if not already open and shows the last match in the file. If an editor is already open then show the previous match in the file.
Remove Selected Matches Removes the currently selected files from the search list.
Remove All Matches Removes all files from the search list.
Run the Current Search Again Repeat the search using the same criteria.
Cancel Current Search Click to cancel the current search in progress.
Show Previous Searches Shows a menu with the 10 previous searches. Select a search to show its results in the view. Select History to manage the list of all previous searches or change the number of histories you can save. Choose Clear History to delete the stored searches.
Pin the Search View Toggles the pin state of the System Search view.
Menu Set filters for the view or open the System Search preference panel.
