Project Directories

You can specify the name of the project directories, where the project files are to be stored.

Figure 1 - Project Directories page

NameFunction

Include

Specifies the name of the include folder, where all the header files for the project are located.

Source

Specifies the name of the source folder, where all the source files for the project are located.

Group Specifies the name of the group folder, where all the group files for the project are located.
Data Specifies the name of the data folder, where all the data files for the project are located.
Install Specifies the name of the Install folder, where all the .sis files for the project are located.
Graphics Specifies the name of the graphics folder, where all the graphic files for the project are located.
