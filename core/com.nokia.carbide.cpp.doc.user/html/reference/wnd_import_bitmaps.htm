Import Bitmaps

Use the Import window to import bitmap resources from the file system.

Figure 1 - Import Bitmaps window

Table 1. Import Bitmaps window—items
Item Explanation
From directory Path to folder containing bitmaps files to import. Click Browse to choose a different location.
Filter Types... Click to open a dialog and select filter options.
Select All Click to select all files.
Deselect All Click to deselect all files.
Into folder Specify the location within the project to store the imported files.
Overwrite existing resources without warning Enable to overwrite existing resources using the same names with the new ones being imported.
Create complete folder structure Create a duplicate of the system folder structure at the project location set in the Into folder text box.
Create selected folders only Create folder structures at the project location set in the Into folder text box only for the selected folders.
