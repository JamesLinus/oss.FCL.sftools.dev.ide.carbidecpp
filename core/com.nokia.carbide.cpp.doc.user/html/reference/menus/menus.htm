Carbide Menus

Carbide has several menu options unique to developing Symbian applications and other shortcuts. Most project related commands are grouped under the Symbian Build option in context menus, while others appear under specific menus like Run or Project.

  • File > New menu options
    • Qt Project
    • Symbian C++ Class
    • Symbian C++ Project
    • Symbian MMP File
  • Right-click object > Symbian Build menu options
    • ABLD Actions
    • Build All Configurations
    • Build PKG File
    • Build Symbian Component
    • Clean Symbian Component
    • Compile
    • Freeze Exports
    • Freeze Symbian Component
    • Preprocess
  • Other menu options
    • Open Command Window
    • Show in Explorer
    • Toggle HW Breakpoint
