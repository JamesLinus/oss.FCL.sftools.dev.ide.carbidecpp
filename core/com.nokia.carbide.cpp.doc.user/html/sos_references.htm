Developer Resources

The following developer references provide information on the S60 SDKs, Symbian SDKs, APIs, and a host of other useful sources. Visit them to enhance your Symbian knowledge.

NOTE WINS (Visual Studio) SDKs are not supported. If you want to do emulation debugging you need a WINSCW SDK.

  • Symbian Foundation (http://www.symbian.org/)
  • Forum Nokia
  • Carbide.c++ IDE and plug-ins discussion forum
  • newlc - a Symbian OS C++ developer portal
  • allaboutsymbian - a community site and portal for phones running the Symbian OS
