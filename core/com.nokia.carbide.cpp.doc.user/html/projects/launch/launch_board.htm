Symbian OS Reference Board Launch

Use the Debug As dialog to select the New Launch Configuration Wizard to create a launch configuration for a development board or reference hardware using a JTAG connection.

Figure 1. Debug As dialog

Creating a Launch Configuration for a Reference Board

  1. Select the Run > Debug menu item, or click the Debug () icon, or press F11

    SHORTCUT Select the Run > Debug As > Debug Symbian OS Application on Reference Board menu item to skip Step 2.

  2. Select the Symbian OS Reference Board Launch item, then click OK
  3. Select a Launch Type, then click Next
  4. Complete each page in the New Launch Configuration Wizard for the chosen Launch Type.
  5. Review the settings in the New Launch Configuration page
  6. Click Finish to download and debug the project on the reference board
