Symbian OS Emulator Launch

Use the Debug As dialog to select the New Launch Configuration Wizard that will create a launch configuration for the project.

Figure 1. Debug As dialog

Creating a Launch Configuration for Emulators

  1. Select the Run > Debug menu item, or click the Debug () icon, or press F11

    SHORTCUT Select the Run > Debug As > Symbian OS Phone or Simulator Launch menu item to skip Step 2.

  2. Select the Symbian OS Emulator Launch item, then click OK
  3. Complete each page in the New Launch Configuration Wizard , then click Next
  4. Review the settings in the New Launch Configuration page
  5. Click Debug to download and debug the project on the emulator
