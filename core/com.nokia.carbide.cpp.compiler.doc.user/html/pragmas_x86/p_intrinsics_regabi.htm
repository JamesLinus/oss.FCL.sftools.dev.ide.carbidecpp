intrinsics_regabi

Controls whether __m64 arguments and __m128{i,d,f} arguments are passed on the stack or via the MMX and SSE registers respectively. The default setting is off.

Syntax

#pragma intrinsics_regabi on | off | reset

Targets

Intel x86
