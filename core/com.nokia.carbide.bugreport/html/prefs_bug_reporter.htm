Bug Reports Preference Panel

Use the Bug Reports preference panel to enter your Carbide Bugzilla user name and password so that the Bug Report plugin submit bug reports in your name. This ensures that you will recieve notice when your bug is addressed by the Carbide development team.


Figure 1 -Bug Reports preference panel

Table 1.Bug Reports preference panel — common items
Item Explanation
Registered login name

Enter the user name you used to register with Carbide Bugzilla.

Password Enter your password for your registered login name.
Send SDK Information If checked, SDK type and version are sent with bug report.
Send Diagnostic Logs If checked, useful debugger event information from current project's debugger log files are sent with report.
Goto Carbide bug reporting site Opens the Carbide Bugzilla home page in your preferred browser. use this to register with the site so you can submit bugs and suggestions using the Bug Reporting plugin.
