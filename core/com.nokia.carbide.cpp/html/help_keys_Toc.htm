Key Shortcuts

The following key shortcuts are available for working with the Carbide.c++ tools:

  • Microsoft Visual Studio Shortcuts
  • Nokia Carbide.c++ Shortcuts
  • Nokia Codewarrior Shortcuts

Exporting shortcuts

Key preferences can be exported using the standard Export menu option:

  1. Click the File > Export... menu option.
  2. Choose General > Preferences in the Select page of the Export window
  3. Click Next
  4. Select Keys Preferences item in the Select Export page
  5. Enter a To preference file name
  6. Click Finish

Importing shortcuts

Key preferences can be imported using the standard Import menu option:

  1. Click the File > Import... menu option.
  2. Choose General > Preferences in the Select page of the Import window
  3. Click Next
  4. Browse or enter the From preference file name in the Select Import page
  5. Click Finish
